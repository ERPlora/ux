Install

pip install ux-jinja

Then add the templates directory to your Jinja loader:

import ux_jinja
env.loader.searchpath.append(str(ux_jinja.TEMPLATES_DIR))

Use a macro

{% raw %}{% from "ui/button.jinja" import button %}
