ERPlora UX

Libreria CSS de componentes para construir aplicaciones de back-office, POS, dashboards, manufactura y recursos humanos. Tema oscuro sobrio por defecto, acento terracota, tipografia Inter. Toda la interactividad va con Datastar — la libreria es 100% declarativa y no requiere paso de compilacion.

Composicion del shell

La documentacion misma es un ejemplo en vivo: la barra lateral usa .ux-sidebar con grupos, items, iconos y badges; el buscador es .ux-search; la cabecera es .ux-topbar con breadcrumb y acciones; la aplicacion entera esta envuelta en .ux-app.

Como empezar

Cuatro pasos para tener la libreria funcionando en cualquier proyecto Jinja2.

Instalacion
pip install ux-jinja — el paquete incluye los macros Jinja2 y las hojas CSS. Sin dependencias de Node.
Ver guia completa
Primer componente
Importa un macro con {% raw %}{% from "ui/card.jinja" import card %}{% endraw %} y listo. Sin configuracion adicional.
Ver card
Integracion con Hotframe
Hotframe ya incluye el loader de macros. Usa reactive() para parches SSE que actualizan senales Datastar desde el servidor.
Demo en vivo
Integracion con Django
Anade el directorio de templates al TEMPLATES de Django y carga la CSS via CDN jsdelivr. Compatible con Django 4+ y Jinja2 backend.
Ver instalacion

Que incluye

Una unica capa de variables CSS define color, tipografia, espaciado, radios, sombras y motion. Temas erplora (oscuro) y erplora-light, mas cinco familias alternativas.
