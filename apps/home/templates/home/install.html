Install

Three steps to start using the @erplora/ux library in any Jinja-rendered project — Flask, FastAPI, Django, Hotframe, anything.

{% call ux_card(title="1. Install the package") %} 
pip install ux-jinja
{% endcall %}
{% call ux_card(title="2. Register the templates loader") %}

Append the bundled ux_jinja.TEMPLATES_DIR to your Jinja loader so the macros under ui/ resolve.

import ux_jinja
env.loader.searchpath.append(str(ux_jinja.TEMPLATES_DIR))
{% endcall %}
{% call ux_card(title="3. Pull the CSS and Datastar from the CDN") %}

Drop these tags in your base template. They are exactly what this site uses.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/ERPlora/ux@main/dist/erplora-ux.min.css" />
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/ERPlora/ux@main/dist/templates/all.min.css" />
<script type="module" src="https://cdn.jsdelivr.net/gh/starfederation/datastar@main/bundles/datastar.js"></script>
{% endcall %}
