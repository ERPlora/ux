Panel Layouts

Los panels son layouts de contenido que van dentro de los shells. Cada seccion ocupa un porcentaje fijo del espacio disponible con scroll interno. Cada seccion tiene un ID unico para integracion con HTMX.

100% Responsive
Scroll interno en cada seccion
Mobile-first (stack vertical)
HTMX ready

Ejemplo Completo: Shell + Panel + HTMX

Uso de shell con sidebar y panel dashboard con integracion HTMX.

<!-- Shell con sidebar + Panel dashboard + HTMX -->
<div class="ux-shell ux-shell--sidebar" x-data="uxShell()">
  <nav class="ux-shell__navbar">...</nav>
  <aside class="ux-shell__sidebar">...</aside>

  <main class="ux-shell__main">
    <div class="ux-panel-dashboard" x-data="uxPanel()">

      <!-- KPIs con HTMX -->
      <section id="kpis" class="ux-panel-dashboard__kpis"
               hx-get="/api/dashboard/kpis"
               hx-trigger="load, refresh">
        <div class="ux-panel-kpi-grid">...</div>
      </section>

      <!-- Charts con HTMX -->
      <section id="charts" class="ux-panel-dashboard__charts"
               hx-get="/api/dashboard/charts"
               hx-trigger="load">...</section>

      <!-- Tabla con header fijo y HTMX -->
      <section id="table" class="ux-panel-dashboard__table">
        <div class="ux-panel-card">
          <div class="ux-panel-card__body ux-panel-card__body--flush">
            <div class="ux-panel-table-wrapper"
                 hx-get="/api/dashboard/table"
                 hx-trigger="load">...</div>
          </div>
        </div>
      </section>

    </div>
  </main>
</div>

<!-- Refrescar seccion especifica -->
<button @click="refresh('kpis')">Actualizar KPIs</button>

Referencia de Clases CSS

Clase Descripcion Responsive
.ux-panel-table-wrapper Tabla con header fijo y scroll sincronizado -
.ux-panel-dashboard Layout KPIs + charts + tabla Charts: column/row
.ux-panel-table Layout acciones + tabla + paginacion -
.ux-panel-split-h Division horizontal 35/65 Stack vertical
.ux-panel-split-v Division vertical 40/60 -
.ux-panel-detail Main + aside con toggle mobile Aside: drawer
.ux-panel-form Header + content + actions -
.ux-panel-settings Menu lateral + content Menu: tabs
.ux-panel-cards Grid de cards responsive 1-2-3-4 cols
.ux-panel-grid-2x2 Grid 2x2 Stack vertical
.ux-panel-grid-1-2 1 arriba + 2 abajo Stack vertical
.ux-panel-grid-2-1 2 arriba + 1 abajo Stack vertical

Componentes Alpine

Componente Metodos Uso
uxPanel() refresh(id), refreshAll() Manejo de HTMX en panels
uxPanelDetail() toggleAside(), openAside(), closeAside() Toggle aside en mobile
