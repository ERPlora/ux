Radio Cards

Tarjetas seleccionables con indicador de check.

Basico
Para usuarios individuales. 5GB de almacenamiento.
Profesional
Para equipos peque単os. 50GB de almacenamiento.
Empresa
Para organizaciones. Almacenamiento ilimitado.
<div class="ux-radio-card-group" x-data="{ selected: 'basic' }">
  <div class="ux-radio-card"
       :class="{ 'ux-radio-card--selected': selected === 'basic' }"
       @click="selected = 'basic'">
    <div class="ux-radio-card__check">
      <svg>...</svg>
    </div>
    <div class="ux-radio-card__title">Titulo</div>
    <div class="ux-radio-card__description">Descripcion</div>
  </div>
</div>

Integracion con HTMX

Radios que sincronizan con el servidor y actualizan contenido dinamicamente.

<!-- Guardar seleccion al cambiar -->
<div class="ux-radio-group">
  <label class="ux-radio">
    <input type="radio" name="theme"
           value="light"
           hx-post="/api/preferences/theme"
           hx-trigger="change"
           hx-swap="none">

    <span class="ux-radio__label">Tema claro</span>
