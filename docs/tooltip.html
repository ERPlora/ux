Tooltips

Tooltips para mostrar información adicional al hacer hover sobre elementos. Útil para explicar iconos, botones y acciones.

Posiciones

Los tooltips pueden aparecer en 4 posiciones: arriba (default), abajo, izquierda y derecha.

Tooltip arriba Tooltip abajo Tooltip izquierda Tooltip derecha 
<!-- Top (default) -->
<span class="ux-tooltip">
  <button class="ux-button">Hover me</button>
  <span class="ux-tooltip__content">Tooltip text</span>
</span>

<!-- Bottom -->
<span class="ux-tooltip ux-tooltip--bottom">...</span>

<!-- Left -->
<span class="ux-tooltip ux-tooltip--left">...</span>

<!-- Right -->
<span class="ux-tooltip ux-tooltip--right">...</span>

Variantes

Diferentes estilos visuales para los tooltips.

Tooltip oscuro (default) Tooltip claro Tooltip primary Tooltip glass 
<!-- Light variant -->

<!-- Primary variant -->
<span class="ux-tooltip ux-tooltip--primary">...</span>

Contenido Multilinea

Tooltips con texto más largo o contenido enriquecido.

Este es un tooltip con texto más largo que se ajusta en múltiples líneas.
Información
Este tooltip tiene un título y descripción para contenido más complejo.
 
<!-- Multiline -->
<span class="ux-tooltip ux-tooltip--multiline">
  <button>Hover</button>
  <span class="ux-tooltip__content">Texto largo...</span>
</span>

<!-- Rich content -->
<span class="ux-tooltip ux-tooltip--rich">
  <button>Hover</button>
  <span class="ux-tooltip__content">
    <div class="ux-tooltip__title">Título</div>
    <div class="ux-tooltip__description">Descripción...</div>
  </span>
</span>

Delay

Retrasar la aparición del tooltip.

Aparece inmediatamente Delay corto (200ms) Delay medio (500ms) Delay largo (1s) 
<span class="ux-tooltip ux-tooltip--delay-short">...</span> <!-- 200ms -->
<span class="ux-tooltip ux-tooltip--delay">...</span>       <!-- 500ms -->
<span class="ux-tooltip ux-tooltip--delay-long">...</span>  <!-- 1000ms -->

Control Programático

Control del tooltip con Alpine.js.

Tooltip controlado 
<div x-data="uxTooltip()">
  <span class="ux-tooltip" :class="{ 'ux-tooltip--open': isOpen }">
    <button @click="toggle()">Click</button>
    <span class="ux-tooltip__content">...</span>
  </span>
</div>

// Métodos disponibles:
show()    // Mostrar tooltip
hide()    // Ocultar tooltip
toggle()  // Alternar

Integracion con HTMX

Los tooltips pueden cargar contenido dinamicamente del servidor usando HTMX.

Tooltip con Contenido Dinamico

Cargar contenido del tooltip desde el servidor al hacer hover.

Cargando... Cargando info... 
<!-- Tooltip que carga contenido al hover -->
<span class="ux-tooltip ux-tooltip--rich">
  <button hx-get="/api/user/1/tooltip"
          hx-target="next .ux-tooltip__content"
          hx-trigger="mouseenter"
          hx-swap="innerHTML">
    Ver usuario
  </button>
  <span class="ux-tooltip__content">
    Cargando...
  </span>
</span>

<!-- Cargar solo una vez con "once" -->
<button hx-get="/api/product/info"
        hx-trigger="mouseenter once">
  Info
</button>

Mostrar preview de datos al hacer hover sobre enlaces o items de lista.

  </button>
  <span class="ux-tooltip__content">
    <!-- El servidor devuelve el preview -->
  </span>
</span>

Tooltip de Ayuda Contextual

Cargar texto de ayuda segun el contexto del formulario.

Formato: ABC-123
Sin simbolo de moneda 
<!-- Tooltip de ayuda que carga del servidor -->
<span class="ux-tooltip">
  <span hx-get="/api/help/field-name"
        hx-target="next .ux-tooltip__content"
        hx-trigger="mouseenter once"
        style="cursor: help">
    <svg>?</svg>
  </span>
  <span class="ux-tooltip__content">
    Cargando ayuda...
  </span>
</span>

Tooltip con Estado en Tiempo Real

Mostrar estado de un proceso que se actualiza periodicamente.

Estado del Servidor
CPU: 45%
RAM: 2.1GB / 4GB
Uptime: 5d 12h
Sincronizando... 12 de 50 items sincronizados 
<!-- Tooltip que se actualiza mientras esta visible -->
<span class="ux-tooltip">
  <button hx-get="/api/server/status"
          hx-target="next .ux-tooltip__content"
          hx-trigger="mouseenter, every 10s[this.matches(':hover')]"
          hx-swap="innerHTML">
    Status
  </button>
  <span class="ux-tooltip__content">
    <!-- Estado actualizado cada 10s mientras hover -->
  </span>
</span>

Tooltip con Accion

Tooltip que incluye acciones que hacen requests HTMX.

<!-- Tooltip con acciones HTMX -->
<span class="ux-tooltip ux-tooltip--rich">
  <button>Item</button>
  <span class="ux-tooltip__content">
    <button hx-post="/api/items/1/edit"
            hx-target="#result">Editar</button>
    <button hx-delete="/api/items/1"
            hx-confirm="Eliminar?">Eliminar</button>
  </span>
</span>
