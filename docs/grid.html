Grid

Sistema de Grid

UX Library usa Bootstrap Grid como sistema de grid externo, con espaciado ajustado al estilo Ionic.

Orden de carga:
1. bootstrap-grid.min.css - Sistema de grid
2. ux-core.js - Sobrescribe --bs-gutter-x/y para espaciado Ionic 
<!-- En tu tema/proyecto -->
<link href="bootstrap-grid.min.css" rel="stylesheet">
<script src="ux-core.js"></script>

<!-- ux-core.js aplica automaticamente: -->
<!-- --bs-gutter-x: 12px (vs 24px de Bootstrap) -->
<!-- .container padding: responsive (16px → 24px → 32px) -->

Referencia de Clases

Bootstrap Grid (externo)

.container / .container-fluidContenedor
.rowFila de columnas
.col / .col-{1-12}Columnas
.col-{sm|md|lg|xl|xxl}-{1-12}Responsive
.offset-{1-11}Offset
.g-{0-5} / .gx-* / .gy-*Gutters
.justify-content-*Alineacion horizontal
.align-items-*Alineacion vertical
