Button

Basico

El boton basico usa la clase .ux-button.

<button class="ux-button">Default</button>
<button class="ux-button ux-button--primary">Primary</button>
<button class="ux-button ux-button--secondary">Secondary</button>
<button class="ux-button ux-button--tertiary">Tertiary</button>

Colores

Todos los colores del sistema disponibles.

<button class="ux-button ux-button--success">Success</button>
<button class="ux-button ux-button--warning">Warning</button>
<button class="ux-button ux-button--danger">Danger</button>

Variantes

Outline, clear, y solid (default).

<button class="ux-button ux-button--primary">Solid</button>
<button class="ux-button ux-button--primary ux-button--outline">Outline</button>
<button class="ux-button ux-button--primary ux-button--clear">Clear</button>

Tamanos

Pequeno, mediano (default), y grande.

<button class="ux-button ux-button--primary ux-button--sm">Small</button>
<button class="ux-button ux-button--primary">Medium</button>
<button class="ux-button ux-button--primary ux-button--lg">Large</button>

Formas

Rounded (default), round, y block.

<button class="ux-button ux-button--primary ux-button--round">Round</button>
<button class="ux-button ux-button--primary ux-button--block">Block</button>

Con Iconos

Iconos al inicio, al final, o solo icono.

<button class="ux-button ux-button--primary">
  <svg class="ux-button__icon">...</svg>
  Add Item
</button>

<button class="ux-button ux-button--primary ux-button--icon">
  <svg>...</svg>
</button>

Estados

Disabled y loading.

<button class="ux-button ux-button--primary" disabled>Disabled</button>
<button class="ux-button ux-button--primary ux-button--loading">Loading</button>

Grupo de Botones

Agrupa botones con .ux-button-group.

<div class="ux-button-group">
  <button class="ux-button ux-button--primary">Left</button>
  <button class="ux-button ux-button--primary">Center</button>
  <button class="ux-button ux-button--primary">Right</button>
</div>

Con HTMX

Uso con atributos hx-*.

<button class="ux-button ux-button--primary"
        hx-post="/api/action"
        hx-target="#result"
        hx-indicator=".htmx-indicator">
  Submit
</button>
