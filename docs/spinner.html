Spinner

Spinner Basico

Indicador de carga circular animado.

<div class="ux-spinner"></div>

Tamanos

Desde extra small hasta extra large.

<div class="ux-spinner ux-spinner--xs"></div>
<div class="ux-spinner ux-spinner--sm"></div>
<div class="ux-spinner"></div>
<div class="ux-spinner ux-spinner--lg"></div>
<div class="ux-spinner ux-spinner--xl"></div>

Colores

Todos los colores del sistema.

<div class="ux-spinner ux-spinner--primary"></div>
<div class="ux-spinner ux-spinner--success"></div>
<div class="ux-spinner ux-spinner--danger"></div>

Spinner Claro (para fondos oscuros)

<div class="ux-spinner ux-spinner--light"></div>
<div class="ux-spinner ux-spinner--white"></div>

Con Texto

Spinner con mensaje de carga.

Cargando... 
<div class="ux-spinner-container">
  <div class="ux-spinner ux-spinner--lg"></div>
  <span class="ux-spinner-container__text">Cargando...</span>
</div>

En Boton

Spinner inline dentro de un boton.

<button class="ux-button ux-button--primary">
  <div class="ux-spinner ux-spinner--xs ux-spinner--white ux-spinner--inline"></div>
  Guardando...
</button>

<button class="ux-button ux-button--primary ux-button--loading">
  Enviar
</button>

Con HTMX

Usar como indicador de HTMX.

<button class="ux-button ux-button--primary"
        hx-post="/api/data"
        hx-indicator="#loading">
  Cargar
</button>

<div id="loading" class="htmx-indicator">
  <div class="ux-spinner"></div>
</div>

Overlay de Pagina Completa

Para bloquear la UI mientras se carga.

  <div class="ux-spinner-container">
    <div class="ux-spinner ux-spinner--xl"></div>
    <span class="ux-spinner-container__text">Procesando...</span>
  </div>
