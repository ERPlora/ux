HTMX + Django

Spinners como indicadores de carga con HTMX y Django.

Template Django con indicador

<!-- templates/partials/lista_datos.html -->
<button class="ux-button ux-color-primary"
        hx-get="{% url 'cargar_datos' %}"
        hx-target="#contenido"
        hx-indicator="#spinner-carga">
  Cargar Datos
</button>

<div id="spinner-carga" class="htmx-indicator">
  <div class="ux-spinner"></div>
</div>

<div id="contenido"></div>

Vista Django

# views.py
import time
from django.shortcuts import render

def cargar_datos(request):
    # Simular carga lenta
    time.sleep(1)

    datos = Modelo.objects.all()
    return render(request, 'partials/datos_lista.html', {
        'datos': datos
    })

Boton con spinner integrado

<!-- Boton que muestra spinner mientras procesa -->
