Img

Uso Basico

Componente de imagen con lazy loading, skeleton mientras carga, y manejo de errores.

Carga el componente:

<script src="components/ux-img.js"></script>

Imagen Simple

La imagen se carga con lazy loading por defecto y muestra un skeleton mientras carga.

<div class="ux-img ux-img--square" style="width: 200px;"
     x-data="uxImg({ src: 'https://example.com/image.jpg' })"
     :class="containerClass">
  <div class="ux-img__skeleton"></div>
  <img class="ux-img__image" :src="isLoaded ? src : ''" :alt="alt">
  <div class="ux-img__error">...</div>
</div>

Lazy Loading

Las imagenes se cargan automaticamente cuando entran en el viewport. Haz scroll para ver las imagenes cargando.

<!-- Lazy loading esta activado por defecto -->
<div x-data="uxImg({ src: '...', lazy: true })">...</div>

<!-- Desactivar lazy loading -->
<div x-data="uxImg({ src: '...', lazy: false })">...</div>

Uso Solo con CSS

Estructura HTML basica sin Alpine.js.

<!-- Imagen ya cargada (agregar clase ux-img--loaded) -->
<div class="ux-img ux-img--md ux-img--rounded ux-img--loaded">
  <img class="ux-img__image" src="image.jpg" alt="">
</div>

<!-- Con skeleton manual -->
<div class="ux-img ux-img--md ux-img--rounded">
  <div class="ux-img__skeleton"></div>
  <img class="ux-img__image"
       src="image.jpg"
       alt=""
       onload="this.parentElement.classList.add('ux-img--loaded')"
       onerror="this.parentElement.classList.add('ux-img--error')">
  <div class="ux-img__error">...</div>
</div>

Integracion con HTMX

El componente funciona con contenido cargado via HTMX.

<!-- El servidor devuelve la imagen con la estructura completa -->
<div hx-get="/api/user/avatar"
     hx-trigger="load"
     hx-swap="innerHTML">
  <!-- Skeleton mientras carga -->
  <div class="ux-img ux-img--md ux-img--circle">
    <div class="ux-img__skeleton"></div>
  </div>
</div>

<!-- Respuesta del servidor -->
<div class="ux-img ux-img--md ux-img--circle ux-img--loaded">
  <img class="ux-img__image" src="/avatars/user.jpg" alt="Usuario">
</div>

Componente Alpine: uxImg()

Propiedades y metodos disponibles.

// Configuracion
uxImg({
  src: 'image.jpg',        // URL de la imagen (requerido)
  alt: 'Descripcion',      // Texto alternativo
  lazy: true,              // Lazy loading (default: true)
  lqip: 'tiny.jpg'         // Low quality image placeholder (blur-up)
})

// Propiedades reactivas
isLoading     // true mientras carga
isLoaded      // true cuando cargo exitosamente
isError       // true si hubo error
state         // 'loading' | 'loaded' | 'error'
containerClass // Clases CSS segun estado

// Metodos
reload()           // Recargar la imagen
setSrc(newSrc)     // Cambiar la imagen

// Eventos
@ux-img:load       // Cuando la imagen cargo
@ux-img:error      // Cuando hubo un error

Ejemplo con eventos:

<div x-data="uxImg({ src: 'image.jpg' })"
     @ux-img:load="console.log('Imagen cargada!')"
     @ux-img:error="console.log('Error al cargar')"
     :class="containerClass">
  ...
</div>

<!-- Cambiar imagen dinamicamente -->
<div x-data="uxImg({ src: '' })" x-ref="img">
  <button @click="$refs.img.__x.$data.setSrc('new-image.jpg')">
    Cargar imagen
  </button>
</div>
