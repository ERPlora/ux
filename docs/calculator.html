Keyboard Support

La calculadora soporta entrada por teclado:

  • 0-9 - Digitos
  • . - Decimal
  • + - * / - Operadores
  • Enter o = - Calcular
  • Escape - Limpiar
  • Backspace - Borrar ultimo digito
  • % - Porcentaje
