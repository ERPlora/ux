Button Group

Agrupacion de botones con bordes conectados y variantes de seleccion.

Basico

Botones agrupados horizontalmente.

<div class="ux-button-group">
  <button class="ux-button">Left</button>
  <button class="ux-button">Center</button>
  <button class="ux-button">Right</button>
</div>

Toggle Group

Seleccion unica con Alpine.js.

<div class="ux-button-group ux-button-group--toggle"
     x-data="uxButtonGroup({ selected: 'week' })">
  <button class="ux-button"
          :class="getButtonClasses('day')"
          @click="select('day')">Day</button>
  ...
</div>

HTMX + Django

Ejemplos de grupos de botones con Django y HTMX para acciones dinamicas.

Filtro con HTMX

Grupo de botones que filtra contenido sin recargar la pagina.

<!-- Template Django: filter_buttons.html -->
<div class="ux-button-group ux-button-group--toggle">
  <button class="ux-button {% if status == 'all' %}ux-button--active{% endif %}"
          hx-get="{% url 'items:list' %}?status=all"
          hx-target="#items-list"
          hx-swap="innerHTML">
    Todos
  </button>
  <button class="ux-button {% if status == 'active' %}ux-button--active{% endif %}"
          hx-get="{% url 'items:list' %}?status=active"
          hx-target="#items-list"
          hx-swap="innerHTML">
    Activos
  </button>
  <button class="ux-button {% if status == 'archived' %}ux-button--active{% endif %}"
          hx-get="{% url 'items:list' %}?status=archived"
          hx-target="#items-list"
          hx-swap="innerHTML">
    Archivados
  </button>
</div>

<div id="items-list">
  {% include 'items/partials/list.html' %}
</div>

Acciones CRUD

Grupo de acciones para un elemento con confirmacion.

<!-- Template Django: item_actions.html -->
<div class="ux-button-group ux-button-group--separated">
  <button class="ux-button"
          hx-get="{% url 'items:edit' item.pk %}"
          hx-target="#modal-content"
          hx-trigger="click">
    Editar
  </button>
  <button class="ux-button"
          hx-post="{% url 'items:duplicate' item.pk %}"
          hx-target="#items-list"
          hx-swap="afterbegin">
    Duplicar
  </button>
  <button class="ux-button ux-color-danger"
          hx-delete="{% url 'items:delete' item.pk %}"
          hx-target="closest .item-row"
          hx-swap="outerHTML swap:1s"
          hx-confirm="Eliminar {{ item.name }}?">
    Eliminar
  </button>
</div>

Vista de Django

# views.py
from django.shortcuts import render
from django.views.decorators.http import require_http_methods

@require_http_methods(["GET"])
def item_list(request):
    status = request.GET.get('status', 'all')

    if status == 'active':
        items = Item.objects.filter(is_active=True)
    elif status == 'archived':
        items = Item.objects.filter(is_archived=True)
    else:
        items = Item.objects.all()

    context = {'items': items, 'status': status}

    # Si es peticion HTMX, devolver solo el partial
    if request.headers.get('HX-Request'):
        return render(request, 'items/partials/list.html', context)

    return render(request, 'items/list.html', context)

Bulk Actions

Acciones masivas para elementos seleccionados.

<!-- Template Django: bulk_actions.html -->
<form id="bulk-form">
  {% csrf_token %}
  <div class="ux-button-group ux-button-group--outline"
       x-data="{ hasSelection: false }"
       @selection-changed.window="hasSelection = $event.detail.count > 0">
    <button type="button"
            class="ux-button"
            :disabled="!hasSelection"
            hx-post="{% url 'items:bulk-archive' %}"
            hx-include="#bulk-form"
            hx-target="#items-list">
      Archivar
    </button>
    <button type="button"
            class="ux-button"
            :disabled="!hasSelection"
            hx-post="{% url 'items:bulk-export' %}"
            hx-include="#bulk-form">
      Exportar
    </button>
    <button type="button"
            class="ux-button ux-color-danger"
            :disabled="!hasSelection"
            hx-delete="{% url 'items:bulk-delete' %}"
            hx-include="#bulk-form"
            hx-target="#items-list"
            hx-confirm="Eliminar elementos seleccionados?">
      Eliminar
    </button>
  </div>
</form>

API

Clase Descripcion
.ux-button-group Contenedor del grupo
.ux-button-group--outline Estilo outline
.ux-button-group--toggle Estilo toggle con seleccion
.ux-button-group--vertical Orientacion vertical
.ux-button-group--block Ancho completo
.ux-button-group--separated Con gap entre botones
.ux-button-group--sm/lg Tamanos
.ux-button-group--glass Efecto glass morphism

Alpine.js API

Propiedad/Metodo Descripcion
selected Valor seleccionado (single)
multiple Permitir seleccion multiple
selectedItems Array de valores (multiple)
select(value) Seleccionar/toggle valor
isSelected(value) Verificar si esta seleccionado
getButtonClasses(value) Clases para el boton
