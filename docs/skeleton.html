Uso Basico

El componente Skeleton proporciona placeholders de carga con animacion shimmer estilo iOS.

Carga el componente:

<script src="components/ux-skeleton.js"></script>

Demo Interactiva

Simula la carga de contenido con el componente Alpine.

<div x-data="{ loading: true }">
  <button @click="loading = !loading">Toggle</button>

  <div class="ux-skeleton-container"
       :class="{ 'ux-skeleton-container--loaded': !loading }">
    <template x-if="loading">
      <!-- Skeleton -->
    </template>
    <template x-if="!loading">
      <!-- Contenido real -->
    </template>
  </div>
</div>

Componente Alpine: uxSkeleton()

Componente simple para manejar estados de carga.

// Uso basico
<div x-data="uxSkeleton()">
  <template x-if="loading">
    <div class="ux-skeleton"></div>
  </template>
  <template x-if="!loading">
    <div>Contenido real</div>
  </template>
</div>

// Configuracion
uxSkeleton({
  loading: true  // Estado inicial (default: true)
})

// Metodos
setLoading(state)  // Cambiar estado de carga
loaded()           // Marcar como cargado (loading = false)

Ejemplo con fetch:

<div x-data="uxSkeleton()" x-init="
  fetch('/api/data')
    .then(r => r.json())
    .then(data => {
      // procesar data
      loaded();
    })
">
  <template x-if="loading">
    <div class="ux-skeleton-card">...</div>
  </template>
  <template x-if="!loading">
    <!-- Contenido real -->
  </template>
</div>

Vista Django que responde al HTMX:

# views.py
from django.shortcuts import render

def user_list(request):
    users = User.objects.all()[:10]
    # Si es peticion HTMX, devolver solo el fragmento
    if request.headers.get('HX-Request'):
        return render(request, 'partials/user_list.html', {'users': users})
    return render(request, 'users/list.html', {'users': users})

Referencia de Clases CSS

Clase Descripcion
.ux-skeleton Clase base con animacion shimmer
.ux-skeleton--text Linea de texto (16px height)
.ux-skeleton--title Titulo (24px height)
.ux-skeleton--circle Forma circular (para avatars)
.ux-skeleton--rect Forma rectangular
.ux-skeleton--rounded Bordes muy redondeados
.ux-skeleton--xs Altura extra pequena (8px)
.ux-skeleton--sm Altura pequena (12px)
.ux-skeleton--md Altura mediana (16px)
.ux-skeleton--lg Altura grande (24px)
.ux-skeleton--xl Altura extra grande (32px)
.ux-skeleton--avatar-* Tamanos de avatar (xs/sm/md/lg/xl)
.ux-skeleton--pulse Animacion pulse en vez de shimmer
.ux-skeleton--no-animation Desactiva la animacion

Patrones Predefinidos

Clase Descripcion
.ux-skeleton-item Item de lista con avatar y texto
.ux-skeleton-card Card con imagen, titulo y texto
.ux-skeleton-profile Perfil con avatar centrado
.ux-skeleton-post Post de red social
.ux-skeleton-table Filas de tabla
.ux-skeleton-form Campos de formulario
.ux-skeleton-chat Mensajes de chat
.ux-skeleton-grid Grid de items cuadrados
.ux-skeleton-container Contenedor con transicion de carga
.ux-skeleton-container--loaded Oculta skeletons con fade out
