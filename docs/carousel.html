Vista Django para slides parciales

# views.py
from django.shortcuts import render

def carousel_slides(request):
    categoria = request.GET.get('categoria', 'destacados')

    if categoria == 'destacados':
        productos = Producto.objects.filter(destacado=True)[:8]
    elif categoria == 'ofertas':
        productos = Producto.objects.filter(en_oferta=True)[:8]
    else:
        productos = Producto.objects.order_by('-created_at')[:8]

    return render(request, 'partials/carousel_slides.html', {
        'productos': productos
    })

API Alpine

Opciones de configuracion

Opcion Default Descripcion
slidesPerView 1 Numero de slides visibles
loop false Navegacion infinita
autoplay false Reproduccion automatica
autoplayInterval 5000 Intervalo en ms
draggable true Permite arrastrar con touch/mouse

Metodos

Metodo Descripcion
prev() Ir al slide anterior
next() Ir al siguiente slide
goTo(index) Ir a un slide especifico
goToPage(page) Ir a una pagina especifica
startAutoplay() Iniciar autoplay
stopAutoplay() Detener autoplay

Propiedades reactivas

Propiedad Descripcion
currentIndex Indice del slide actual
slidesCount Total de slides
canGoPrev Si puede ir atras
canGoNext Si puede ir adelante
translateX Valor de traduccion CSS
paginationDots Array para generar dots

Eventos

Evento Detail
ux-carousel:init { slidesCount, currentIndex }
ux-carousel:change { index, direction }
