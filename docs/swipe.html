Swipe & Gestures

Directivas de Gestos

Directivas Alpine.js para gestos tactiles: swipe, drag, pull-to-refresh, long-press, pinch y tap.

<script src="components/ux-swipe.js"></script>

x-swipe

Detecta swipes en cualquier direccion. Usa modificadores para filtrar direccion.

Swipe aqui 
<!-- Detectar cualquier swipe -->
<div x-swipe="handleSwipe">...</div>

<!-- Solo swipe izquierda -->
<div x-swipe.left="handleLeft">...</div>

<!-- Solo swipe derecha -->
<div x-swipe.right="handleRight">...</div>

<!-- Callback recibe: { direction, distance, velocity, angle } -->

Swipe por Direccion

Usando modificadores .left, .right, .up, .down para filtrar.

Solo izquierda/derecha 
<div
     x-swipe.left="() => action = 'delete'"
     x-swipe.right="() => action = 'archive'">
  ...
</div>

Swipeable List Item

Componente para items de lista con acciones al deslizar (estilo iOS Mail).

JD
John Doe
Desliza para ver acciones
<div x-data="uxSwipeItem({ endActionsWidth: 160 })" class="ux-swipe-container">
  <!-- Acciones (se revelan al deslizar) -->
  <div class="ux-swipe-actions ux-swipe-actions--end">
    <button class="ux-swipe-action ux-swipe-action--more">...</button>
    <button class="ux-swipe-action ux-swipe-action--delete">...</button>
  </div>

  <!-- Contenido -->
  <div class="ux-swipe-content" :style="contentStyle">
    ...
  </div>
</div>

x-long-press

Detecta pulsacion larga. Por defecto 500ms, configurable con modificador.

Manten presionado 
<!-- Long press default (500ms) -->
<div x-long-press="handleLongPress">...</div>

<!-- Long press 800ms -->
<div x-long-press.800ms="handleLongPress">...</div>

<!-- Long press 1 segundo -->
<div x-long-press.1000ms="handleLongPress">...</div>

x-tap

Detecta taps simples o dobles. Diferente de click, es especifico para touch.

Tap / Double Tap 
<!-- Single tap -->
<div x-tap="handleTap">...</div>

<!-- Double tap -->
<div x-tap.double="handleDoubleTap">...</div>

<!-- Ambos -->
<div x-tap="single" x-tap.double="double">...</div>

x-drag

Permite arrastrar elementos. Recibe eventos start, move, end.

Drag 
<div x-drag="handleDrag">...</div>

<!-- Handler recibe: -->
// { type: 'start', startPos, offset, element }
// { type: 'move', startPos, currentPos, delta: {x, y}, element }
// { type: 'end', startPos, endPos, delta: {x, y}, element }

Drag Handle

Estilos para handle de arrastre en listas reordenables.

Item con drag handle
Otro item 
<div class="ux-drag-handle">
  <svg class="ux-drag-handle__icon">...</svg>
</div>

x-pull-refresh

Pull to refresh para listas. Requiere contenedor con scroll.

Desliza hacia abajo desde el inicio de la lista para refrescar (mejor en movil)

<div x-pull-refresh="async () => await fetchData()"
     style="height: 400px; overflow-y: auto;">
  <div class="ux-pull-content">
    <!-- Lista de items -->
  </div>
</div>

x-pinch

Detecta gesto de pinch (zoom) con dos dedos. Solo funciona en touch.

Pinch to Zoom 
<div x-pinch="handlePinch">...</div>

<!-- Handler recibe: -->
// { type: 'pinch', scale, delta, center: {x, y} }

API - Directivas Alpine

Directiva Modificadores Descripcion
x-swipe .left .right .up .down Detecta swipe en direccion
x-drag .handle:selector Drag & drop con handle opcional
x-pull-refresh - Pull to refresh para listas
x-long-press .{n}ms Long press con duracion configurable
x-pinch - Pinch zoom (solo touch)
x-tap .double Single o double tap

API - Clases CSS

Clase Descripcion
.ux-swipe-container Contenedor para swipe item
.ux-swipe-content Contenido principal que se desliza
.ux-swipe-actions Contenedor de acciones
.ux-swipe-actions--start Acciones al inicio (izquierda)
.ux-swipe-actions--end Acciones al final (derecha)
.ux-swipe-action Boton de accion individual
.ux-drag-handle Handle para drag en listas
.ux-draggable Elemento arrastrable
.ux-drop-zone Zona de drop
.ux-pull-indicator Indicador de pull refresh

API - uxSwipeItem()

Componente Alpine para swipeable list items.

Propiedad/Metodo Descripcion
offset Desplazamiento actual en px
isSwiping true durante swipe activo
openSide 'start', 'end' o null
contentStyle Getter para :style del contenido
openStart() Abre acciones del inicio
openEnd() Abre acciones del final
close() Cierra las acciones 
// Configuracion
uxSwipeItem({
  startActionsWidth: 160,  // Ancho acciones izquierda
  endActionsWidth: 80,     // Ancho acciones derecha
  threshold: 40            // Distancia para activar snap
})
