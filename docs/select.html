Select Personalizado (Alpine)

Select con dropdown personalizado usando Alpine.js.

Framework 
<div class="ux-select"
     x-data="uxSelect({
       options: [
         { value: 'opt1', label: 'Opcion 1' },
         { value: 'opt2', label: 'Opcion 2' }

       placeholder: 'Selecciona...'
     })">

</div>

Search Select (Autocomplete)

Select con busqueda integrada. Ideal para listas largas como paises, ciudades o timezones. El dropdown aparece al escribir 2+ caracteres.

Escribe al menos 2 caracteres para buscar 
<div class="ux-search-select"
     x-data="uxSearchSelect({
       options: [
         { value: 'Europe/Madrid', label: '(UTC+01:00) Madrid', description: 'Central European Time' },
         { value: 'America/New_York', label: '(UTC-05:00) New York', description: 'Eastern Standard Time' },
         // ... mas opciones
       ],
       placeholder: 'Escribe para buscar...',
       minChars: 2
     })">
  <label class="ux-search-select__label">Timezone</label>
  <div class="ux-search-select__wrapper">
    <div class="ux-search-select__input-wrapper">
      <span class="ux-search-select__icon">...</span>
      <input type="text"
             class="ux-search-select__input"
             x-model="query"
             @input="handleInput()"
             @keydown="handleKeydown($event)">
      <button class="ux-search-select__clear" @click="clear()">...</button>
    </div>
    <div class="ux-search-select__dropdown">
      <template x-for="option in filteredOptions">
        <div class="ux-search-select__option" @click="selectOption(option)">
          <span x-html="highlightMatch(option.label)"></span>
        </div>
      </template>
    </div>
  </div>
</div>

API Alpine

Metodos disponibles en el componente.

// uxSelect
toggle()        // Abre/cierra el dropdown
open()          // Abre el dropdown
close()         // Cierra el dropdown
select(option)  // Selecciona una opcion
reset()         // Resetea el valor
validate()      // Valida el campo

// Propiedades
value           // Valor seleccionado
displayValue    // Label de la opcion seleccionada
isOpen          // Estado del dropdown
error           // Mensaje de error

// uxMultiSelect (adicionales)
values          // Array de valores seleccionados
toggleOption()  // Toggle una opcion
removeValue()   // Elimina un valor

// uxSearchSelect (autocomplete)
query           // Texto de busqueda actual
value           // Valor seleccionado
filteredOptions // Opciones filtradas por query
handleInput()   // Maneja cambios en el input
selectOption()  // Selecciona una opcion
clear()         // Limpia seleccion y query
highlightMatch()// Resalta texto coincidente
validate()      // Valida el campo
reset()         // Resetea todo

// Config uxSearchSelect
minChars: 2     // Caracteres minimos para buscar
maxResults: 50  // Maximo resultados mostrados
emptyText       // Texto cuando no hay resultados
