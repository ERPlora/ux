Quick Start Componentes

Quick Start

Carga solo los componentes que necesitas:

<!-- Dependencias opcionales -->
<script defer src="https://unpkg.com/alpinejs@3.x.x/dist/cdn.min.js"></script>
<script src="https://unpkg.com/htmx.org@1.x.x"></script>

<!-- Core (requerido) -->
<script src="components/ux-core.js"></script>

<!-- Componentes individuales -->
<script src="components/ux-button.js"></script>
<script src="components/ux-modal.js"></script>

<!-- O cargar todos -->
<script src="ux-all.js"></script>

Sistema de Colores (3 Capas)

Arquitectura de colores escalable y mantenible. Los plugins solo deben usar tokens semánticos (Capa 2).

Capa 1: Primitivos

Colores base (uso interno)

--ux-gray-{50-950}
--ux-blue-{50-900}
--ux-red-{50-900}
--ux-green-{50-900}
--ux-amber-{50-900}
--ux-purple-{50-900}
--ux-teal-{50-900}
+ cyan, indigo, orange, rose, violet, emerald, slate

Capa 2: Semánticos

Usar estos

Tokens con significado

--ux-primary + shade/tint
--ux-secondary
--ux-success
--ux-warning
--ux-danger
--ux-text + secondary/tertiary
--ux-surface + secondary
--ux-border-color

Sistema de Grid

Para layouts complejos con columnas responsive, recomendamos usar Bootstrap Grid (solo CSS, ~7KB gzipped):

<!-- Solo el grid de Bootstrap, sin JS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap-grid.min.css">

Ejemplo de uso con UX Library:

<div class="container">
  <div class="row g-3">
    <div class="col-12 col-md-6 col-lg-4">
      <div class="ux-card">Card 1</div>
    </div>
    <div class="col-12 col-md-6 col-lg-4">
      <div class="ux-card">Card 2</div>
    </div>
    <div class="col-12 col-md-12 col-lg-4">
      <div class="ux-card">Card 3</div>
    </div>
  </div>
</div>

CSS Grid Nativo

.ux-css-grid - display: grid
.ux-css-grid--{2|3|4|6|12}
.ux-css-grid--auto-fit / --auto-fill
.ux-gap-{xs|sm|md|lg|xl}

<div class="ux-css-grid ux-css-grid--3 ux-gap-md">

  <div>Item 2</div>
  <div>Item 3</div>
</div>
