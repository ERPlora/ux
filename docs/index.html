UX - Universal eXtensions

Zero-dependency, framework-agnostic CSS component library con estilo iOS. Funciona con HTMX, React, Vue, o JavaScript vanilla.

Quick Start

<!-- CSS (requerido) -->
<link rel="stylesheet" href="dist/ux.min.css">

<!-- JS Helpers (opcional) -->
<script src="dist/ux.min.js"></script>

<!-- Uso básico -->
<button class="ux-button ux-color-primary">Button</button>

<!-- Modal con data-state -->
<div class="ux-modal-backdrop" data-state="closed">
  <div class="ux-modal">...</div>
</div>

Bootstrap Grid

UX usa Bootstrap Grid 5.3 para el sistema de layout (opcional):

<!-- Bootstrap Grid (solo grid, sin componentes) -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap-grid.min.css" rel="stylesheet">

<!-- UX CSS -->
<link rel="stylesheet" href="dist/ux.min.css">

<!-- UX JS (opcional) -->
<script src="dist/ux.min.js"></script>

Crear Tema de Colores

Crea tu propio tema sobrescribiendo las variables CSS del primary color:

/* Mi tema personalizado */
.ux-theme-custom {
  /* Color primario - usar tonos de tu marca */
  --ux-primary: #8B5CF6;           /* Violeta */
  --ux-primary-rgb: 139, 92, 246;
  --ux-primary-contrast: #ffffff;
  --ux-primary-shade: #7C3AED;     /* Más oscuro */
  --ux-primary-tint: #A78BFA;      /* Más claro */
}
<!-- Aplicar a todo el documento -->
<body class="ux-theme-custom">

Integración con HTMX + Django

UX está optimizado para trabajar con HTMX y Django. Ejemplo de configuración:

<!-- base.html - Template base Django -->
{% load static %}
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap-grid.min.css" rel="stylesheet">
  <link rel="stylesheet" href="{% static 'css/ux.min.css' %}">
  <script src="{% static 'js/ux.min.js' %}"></script>
  <script src="https://unpkg.com/htmx.org@2.0.4"></script>
</head>
<body class="ux-app" hx-headers='{"X-CSRFToken": "{{ csrf_token }}"}'>
  {% block content %}{% endblock %}
</body>
</html>

Ejemplo de vista Django con HTMX:

# views.py
from django.shortcuts import render
from django.http import HttpResponse

def lista_usuarios(request):
    usuarios = Usuario.objects.all()

    # Si es petición HTMX, retornar solo el fragmento
    if request.headers.get('HX-Request'):
        return render(request, 'partials/usuarios_lista.html', {
            'usuarios': usuarios
        })

    return render(request, 'usuarios/lista.html', {
        'usuarios': usuarios
    })

Template con componentes UX y HTMX:

<!-- templates/usuarios/lista.html -->
{% extends 'base.html' %}
{% block content %}
<div class="container ux-p-lg">
  <div class="ux-card">
    <div class="ux-card__header">
      <h2 class="ux-card__title">Usuarios</h2>
      <button class="ux-button ux-color-primary"
              hx-get="{% url 'usuario_crear' %}"
              hx-target="#modal-container"
              hx-swap="innerHTML">
        Nuevo Usuario
      </button>
    </div>
    <div class="ux-card__content">
      <div id="usuarios-lista"
           hx-get="{% url 'usuarios_lista' %}"
           hx-trigger="load"
           hx-swap="innerHTML">
        <div class="ux-spinner"></div>
      </div>
    </div>
  </div>
</div>
<div id="modal-container"></div>
{% endblock %}
