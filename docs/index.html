Quick Start Componentes

Quick Start

Carga solo los componentes que necesitas:

<!-- Dependencias opcionales -->
<script defer src="https://unpkg.com/alpinejs@3.x.x/dist/cdn.min.js"></script>
<script src="https://unpkg.com/htmx.org@1.x.x"></script>

<!-- Core (requerido) -->
<script src="components/ux-core.js"></script>

<!-- Componentes individuales -->
<script src="components/ux-button.js"></script>
<script src="components/ux-modal.js"></script>

<!-- O cargar todos -->
<script src="ux-all.js"></script>

Clases Utility (ux-core.js)

Clases CSS incluidas en el core para uso rapido sin componentes adicionales.

Flexbox

.ux-flex - display: flex
.ux-flex-col - flex-direction: column
.ux-flex-wrap - flex-wrap: wrap
.ux-items-center - align-items: center
.ux-justify-center - justify-content: center
.ux-justify-between - space-between
.ux-gap-{xs|sm|md|lg} - gap spacing

Spacing

.ux-p-{size} - padding all sides
.ux-m-{size} - margin all sides
.ux-px-{size} - padding horizontal
.ux-py-{size} - padding vertical
.ux-mt-{size} - margin-top
.ux-mb-{size} - margin-bottom
.ux-mx-auto - margin horizontal auto

Typography

.ux-text-{xs|sm|base|md|lg|xl|2xl|3xl}
.ux-text-center - text-align: center
.ux-font-{normal|medium|semibold|bold}
.ux-text-primary - color primary
.ux-text-secondary - color secondary
.ux-text-success - color success

Display & Effects

.ux-hidden - display: none
.ux-block - display: block
.ux-rounded-{sm|lg|xl|full}
.ux-shadow-{sm|md|lg|xl}
.ux-ripple - ripple effect on tap
.ux-tap - scale down on press
