Menu

Menus desplegables y contextuales con soporte para iconos, atajos de teclado, estados y submenus.

<script src="components/ux-menu.js"></script>

Uso Basico

Menu desplegable con items simples.

  • Nuevo archivo
  • Abrir...
  • Guardar
  • Cerrar
<div class="ux-menu" x-data="{ open: false }" :class="{ 'ux-menu--open': open }">
  <button class="ux-button" @click="open = !open">Opciones</button>
  <div class="ux-menu__content ux-menu__content--bottom">
    <ul class="ux-menu__list">
      <li><a href="#" class="ux-menu__item">Nuevo archivo</a></li>
      <li><a href="#" class="ux-menu__item">Abrir...</a></li>
      <li><a href="#" class="ux-menu__item">Guardar</a></li>
      <li class="ux-menu__divider"></li>
      <li><a href="#" class="ux-menu__item">Cerrar</a></li>
    </ul>
  </div>
</div>

Componente Alpine

Usa el componente uxMenu para control completo con navegacion por teclado.

<div x-data="uxMenu({ position: 'bottom-left' })"
     :class="{ 'ux-menu--open': isOpen }"
     @keydown="handleKeydown($event)">

  <button @click="toggle()">Menu</button>

  <div class="ux-menu__content" x-ref="content">
    <ul class="ux-menu__list">
      <template x-for="item in items">
        <li>
          <a class="ux-menu__item"
             :class="{ 'ux-menu__item--selected': isSelected(item) }"
             @click="select(item)">
            <span x-text="item.label"></span>
          </a>
        </li>
      </template>
    </ul>
  </div>
</div>

<script>
// Configuracion del componente
uxMenu({
  position: 'bottom-left',  // Posicion del menu
  items: [...],             // Array de items
  selectedValue: null       // Valor seleccionado inicial
})

// Metodos disponibles
open()           // Abrir menu
close()          // Cerrar menu
toggle()         // Alternar estado
select(item)     // Seleccionar item
isSelected(item) // Verificar si esta seleccionado
handleKeydown()  // Manejo de teclado (Escape, Arrow keys)
</script>
