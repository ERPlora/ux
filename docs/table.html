Variables CSS

Variable Default Descripcion
--ux-table-border-color var(--ux-border-color) Color de bordes
--ux-table-header-bg var(--ux-surface-secondary) Fondo del header
--ux-table-header-color var(--ux-text) Color texto header
--ux-table-cell-padding var(--ux-space-sm) var(--ux-space-md) Padding de celdas
--ux-table-stripe-bg var(--ux-surface-secondary) Fondo filas alternadas
--ux-table-hover-bg rgba(var(--ux-primary-rgb), 0.05) Fondo hover
--ux-table-border-radius var(--ux-radius-lg) Radio del wrapper

Referencia de Clases

Clase Descripcion
.ux-table Clase base para tablas
.ux-table-wrapper Wrapper con bordes redondeados
.ux-table-responsive Contenedor con scroll horizontal
.ux-table--striped Filas con fondo alternado
.ux-table--hover Efecto hover en filas
.ux-table--bordered Bordes en todas las celdas
.ux-table--compact Padding reducido
.ux-table--relaxed Padding aumentado
.ux-table--fixed Ancho de columnas fijo
.ux-table--glass Variante glass morphism
.ux-table-wrapper--glass Wrapper glass
.text-center Alinear texto al centro
.text-right Alinear texto a la derecha
.text-nowrap Evitar wrap del texto
