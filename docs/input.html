Input Basico

Campo de texto simple con la clase .ux-input.

<input type="text" class="ux-input" placeholder="Escribe algo...">

Con Label

Usa .ux-input-group para agrupar label e input.

Nunca compartiremos tu email. 
<div class="ux-input-group">
  <label class="ux-input__label">Email</label>
  <input type="email" class="ux-input" placeholder="tu@email.com">
  <span class="ux-input__helper">Texto de ayuda</span>
</div>

Label Flotante

Estilo iOS con label que flota al enfocar.

<div class="ux-input-group ux-input-group--floating">
  <input type="text" class="ux-input" placeholder=" ">
  <label class="ux-input__label">Nombre</label>
</div>

Password con Toggle

Input de contrasena con boton para mostrar/ocultar.

<div x-data="uxPassword()">
  <div class="ux-input-password">
    <input :type="inputType" class="ux-input" x-model="value">
    <button class="ux-input-password__toggle" @click="toggle()">
      <svg x-show="!showPassword">...</svg>
      <svg x-show="showPassword">...</svg>
    </button>
  </div>
</div>

Tipos de Input

Diferentes tipos HTML nativos estilizados.

Con Iconos

Iconos al inicio o al final del input.

<div class="ux-input-wrapper ux-input-wrapper--icon-start">
  <svg class="ux-input__icon">...</svg>
  <input type="text" class="ux-input">
</div>

Estados de Validacion

Success, error, y warning.

Email verificado correctamente.
Este campo es obligatorio.
Usa al menos 8 caracteres. 
<div class="ux-input-group ux-input-group--error">
  <label class="ux-input__label">Campo</label>
  <input class="ux-input">
  <span class="ux-input__error">Mensaje de error</span>
</div>

Disabled y Readonly

Tamanos

<input class="ux-input ux-input--sm">
<input class="ux-input">
<input class="ux-input ux-input--lg">

Variantes

Filled, outline, y underline.

<input class="ux-input ux-input--filled">
<input class="ux-input ux-input--outline">
<input class="ux-input ux-input--underline">

Textarea

Area de texto multilinea.

<textarea class="ux-input" placeholder="Mensaje..."></textarea>

Con Addon

Prefijos y sufijos para inputs.

https://
.00 
<div class="ux-input-group ux-input-group--horizontal">
  <span class="ux-input-group__addon">https://</span>

</div>

Con HTMX

Validacion en tiempo real con HTMX.

<input type="email"
       class="ux-input"
       name="email"
       hx-post="/validate/email"
       hx-trigger="blur"
       hx-target="next .ux-input__error">
<span class="ux-input__error"></span>
