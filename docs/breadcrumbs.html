Breadcrumbs Colapsables (Alpine.js)

Para rutas muy largas, colapsa los items intermedios automaticamente.

Con ellipsis y dropdown
<nav class="ux-breadcrumbs"
     x-data="uxBreadcrumbs({
       items: [
         { label: 'Inicio', href: '#' },
         { label: 'Documentos', href: '#' },
         { label: 'Proyectos', href: '#' },
         { label: '2024', href: '#' },
         { label: 'Reportes', href: '#' },
         { label: 'Enero', href: '#' },
         { label: 'Reporte Final', href: '#', current: true }
       ],
       maxItems: 4
     })">
  <!-- Template items automaticamente -->
</nav>

Referencia de Clases CSS

Clase Descripcion
.ux-breadcrumbs Contenedor principal
.ux-breadcrumb Item individual
.ux-breadcrumb__link Link o texto del item
.ux-breadcrumb__separator Separador entre items
.ux-breadcrumb__icon Contenedor de icono
.ux-breadcrumb--current Marca el item como pagina actual
.ux-breadcrumb--disabled Item deshabilitado
.ux-breadcrumbs--slash Separador tipo slash (/)
.ux-breadcrumbs--arrow Separador tipo flecha (>)
.ux-breadcrumbs--dot Separador tipo punto
.ux-breadcrumbs--contained Fondo con surface-secondary
.ux-breadcrumbs--outlined Con borde
.ux-breadcrumbs--sm Tamano pequeno
.ux-breadcrumbs--lg Tamano grande
.ux-breadcrumbs--responsive Oculta items intermedios en mobile

Componente Alpine

uxBreadcrumbs(config) - Para breadcrumbs colapsables

// Opciones de configuracion
{
  items: [
    { label: 'Inicio', href: '#' },
    { label: 'Seccion', href: '#' },
    { label: 'Pagina', href: '#', current: true }
  ],
  maxItems: 4  // Colapsa si hay mas de 4 items
}

// Propiedades reactivas
visibleItems    // Items a mostrar (con ellipsis si colapsado)
collapsedItems  // Items ocultos en el dropdown

// Metodos
expand()        // Expande todos los items
toggleDropdown() // Abre/cierra el dropdown
closeDropdown()  // Cierra el dropdown
isLast(index)   // Verifica si es el ultimo item
