Alert Dialog Basico

Dialogo de alerta estilo iOS con titulo, mensaje y botones.

<div x-data="uxAlert()">
  <button @click="open({ title: 'Titulo', message: 'Mensaje' })">Abrir</button>

  <template x-teleport="body">
    <div class="ux-alert-backdrop" :class="isOpen && 'ux-alert-backdrop--open'">
      <div class="ux-alert">
        <div class="ux-alert__content">
          <h2 class="ux-alert__title" x-text="title"></h2>
          <p class="ux-alert__message" x-text="message"></p>
        </div>
        <div class="ux-alert__buttons">
          <button class="ux-alert__button" @click="close()">OK</button>
        </div>
      </div>
    </div>
  </template>
</div>

HTMX + Django

Ejemplos de alertas dinamicas con HTMX y Django para notificaciones del servidor.

Alert de confirmacion antes de eliminar

Mostrar un dialog de confirmacion antes de ejecutar una accion destructiva.

<!-- Boton que dispara el alert de confirmacion -->
<button class="ux-button ux-button ux-color-danger"
        x-data="{ showConfirm: false }"
        @click="showConfirm = true">
  Eliminar Producto

  <template x-teleport="body">
    <div class="ux-alert-backdrop" :class="showConfirm && 'ux-alert-backdrop--open'">
      <div class="ux-alert">
        <div class="ux-alert__content">
          <h2 class="ux-alert__title">Eliminar Producto</h2>
          <p class="ux-alert__message">Esta accion no se puede deshacer.</p>
        </div>
        <div class="ux-alert__buttons">
          <button class="ux-alert__button" @click="showConfirm = false">Cancelar</button>
          <button class="ux-alert__button ux-alert__button--destructive"
                  hx-delete="/productos/{{ producto.id }}/"
                  hx-target="#product-list"
                  hx-swap="outerHTML"
                  @click="showConfirm = false">
            Eliminar
          </button>
        </div>
      </div>
    </div>
  </template>
</button>

Alert modal desde respuesta HTMX

Cargar contenido de alert dinamicamente desde el servidor.

<!-- Boton que carga el alert del servidor -->
<button class="ux-button"
        hx-get="/api/confirm-action/"
        hx-target="body"
        hx-swap="beforeend">
  Accion con Confirmacion
</button>
# views.py
def confirm_action(request):
    return HttpResponse('''
        <div x-data="{ open: true }" x-init="$watch('open', v => !v && $el.remove())">
            <div class="ux-alert-backdrop" :class="open && 'ux-alert-backdrop--open'">
                <div class="ux-alert">
                    <div class="ux-alert__content">
                        <h2 class="ux-alert__title">Confirmacion Requerida</h2>
                        <p class="ux-alert__message">El servidor solicita tu confirmacion.</p>
                    </div>
                    <div class="ux-alert__buttons">
                        <button class="ux-alert__button" @click="open = false">Cancelar</button>
                        <button class="ux-alert__button ux-alert__button--primary"
                                hx-post="/api/execute-action/"
                                hx-swap="none"
                                @click="open = false">
                            Confirmar
                        </button>
                    </div>
                </div>
            </div>
        </div>
    ''')

Usando HX-Trigger para mostrar alertas

El servidor puede disparar eventos personalizados via headers.

# views.py
from django.http import HttpResponse

def crear_producto(request):
    # Crear producto...
    response = HttpResponse("Producto creado")
    response['HX-Trigger'] = json.dumps({
        'showAlert': {
            'type': 'success',
            'title': 'Producto Creado',
            'message': 'El producto se creo exitosamente.'
        }
    })
    return response
<!-- Listener global para alertas -->
<div x-data="{
       alert: null,
       show: false
     }"
     @show-alert.window="alert = $event.detail; show = true">

  <template x-teleport="body">
    <div class="ux-alert-backdrop" :class="show && 'ux-alert-backdrop--open'">
      <div class="ux-alert">
        <div class="ux-alert__content">
          <div class="ux-alert__icon" :class="'ux-alert__icon--' + alert?.type">...</div>
          <h2 class="ux-alert__title" x-text="alert?.title"></h2>
          <p class="ux-alert__message" x-text="alert?.message"></p>
        </div>
        <div class="ux-alert__buttons">
          <button class="ux-alert__button ux-alert__button--primary" @click="show = false">OK</button>
        </div>
      </div>
    </div>
  </template>
</div>

<script>
  // Convertir HX-Trigger a evento Alpine
  document.body.addEventListener('showAlert', (e) => {
    window.dispatchEvent(new CustomEvent('show-alert', { detail: e.detail }));
  });
</script>

API Alpine - uxAlert()

Metodo/Propiedad Descripcion
isOpen Boolean - Estado del alert
title String - Titulo del alert
message String - Mensaje del alert
open(options) Abre el alert con opciones { title, message, buttons, inputs }
close() Cierra el alert
confirm(title, msg) Retorna Promise<boolean>
prompt(title, msg, placeholder) Retorna Promise<string|null>

API Alpine - uxAlertBanner()

Metodo/Propiedad Descripcion
visible Boolean - Visibilidad del banner
dismissible Boolean - Si se puede cerrar (default: true)
dismiss() Oculta el banner
show() Muestra el banner
