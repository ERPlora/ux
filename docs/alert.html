Alert

Alert Dialog Basico

Dialogo de alerta estilo iOS con titulo, mensaje y botones.

<div x-data="uxAlert()">
  <button @click="open({ title: 'Titulo', message: 'Mensaje' })">Abrir</button>

  <template x-teleport="body">
    <div class="ux-alert-backdrop" :class="isOpen && 'ux-alert-backdrop--open'">
      <div class="ux-alert">
        <div class="ux-alert__content">
          <h2 class="ux-alert__title" x-text="title"></h2>
          <p class="ux-alert__message" x-text="message"></p>
        </div>
        <div class="ux-alert__buttons">
          <button class="ux-alert__button" @click="close()">OK</button>
        </div>
      </div>
    </div>
  </template>
</div>

Alert con Icono

Alertas con iconos para diferentes estados: success, warning, danger, info.

<div class="ux-alert__icon ux-alert__icon--success">
  <svg>...</svg>
</div>
<!-- Variantes: --success, --warning, --danger, --info -->

Confirm Dialog

Alert con botones de confirmacion y cancelacion.

<div class="ux-alert__buttons">
  <button class="ux-alert__button">Cancelar</button>
  <button class="ux-alert__button ux-alert__button--primary">Aceptar</button>
</div>

Accion Destructiva

Alert para confirmar acciones peligrosas como eliminar.

<button class="ux-alert__button ux-alert__button--cancel">Cancelar</button>
<button class="ux-alert__button ux-alert__button--destructive">Eliminar</button>

Botones Apilados

Para multiples opciones, los botones se pueden apilar verticalmente.

<div class="ux-alert__buttons ux-alert__buttons--stacked">
  <button class="ux-alert__button ux-alert__button--primary">Opcion A</button>
  <button class="ux-alert__button">Opcion B</button>
  <button class="ux-alert__button ux-alert__button--destructive">Cancelar</button>
</div>

Prompt con Input

Alert con campo de entrada para solicitar informacion al usuario.

Hola, ! 
<div class="ux-alert__inputs">
  <input type="text" class="ux-alert__input" placeholder="Nombre" x-model="value">
</div>

Login Prompt

Alert con multiples campos de entrada.

<div class="ux-alert__inputs">
  <input type="text" class="ux-alert__input" placeholder="Usuario">
  <input type="password" class="ux-alert__input" placeholder="Contrasena">
</div>

Alert Ancho (Estilo Android)

Variante mas ancha con botones alineados a la derecha.

<div class="ux-alert ux-alert--wide">...</div>

Alert Banner (Inline)

Alertas inline para mostrar mensajes dentro del contenido.

Exito

Tu archivo ha sido guardado correctamente.

Advertencia

Tu sesion expirara en 5 minutos.

Error

No se pudo completar la operacion.

Informacion

Hay una nueva version disponible.

<div class="ux-alert-banner ux-alert-banner--success" x-data="uxAlertBanner()">
  <div class="ux-alert-banner__icon">...</div>
  <div class="ux-alert-banner__content">
    <h4 class="ux-alert-banner__title">Exito</h4>
    <p class="ux-alert-banner__message">Mensaje</p>
  </div>
  <button class="ux-alert-banner__close" @click="dismiss()">X</button>
</div>
<!-- Variantes: --success, --warning, --danger, --info -->

API Alpine - uxAlert()

Metodo/Propiedad Descripcion
isOpen Boolean - Estado del alert
title String - Titulo del alert
message String - Mensaje del alert
open(options) Abre el alert con opciones { title, message, buttons, inputs }
close() Cierra el alert
confirm(title, msg) Retorna Promise<boolean>
prompt(title, msg, placeholder) Retorna Promise<string|null>

API Alpine - uxAlertBanner()

Metodo/Propiedad Descripcion
visible Boolean - Visibilidad del banner
dismissible Boolean - Si se puede cerrar (default: true)
dismiss() Oculta el banner
show() Muestra el banner
