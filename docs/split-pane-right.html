Split Pane Right

Panel lateral derecho con toggle. En desktop el panel es visible y puede colapsarse. En mobile aparece como overlay desde la derecha.

Comportamiento

  • Desktop (>= 992px): Panel visible, puede ocultarse con toggle
  • Mobile (< 992px): Panel como overlay desde la derecha con backdrop
Main Content

Este es el contenido principal. El panel lateral puede mostrarse u ocultarse con el boton de la barra de navegacion.

API

Metodo/Prop Descripcion
toggle() Toggle panel (desktop: collapse, mobile: overlay)
open() Abrir overlay (mobile)
close() Cerrar overlay (mobile)
show() Mostrar panel (desktop)
hide() Ocultar panel (desktop)
isOpen Estado del overlay (mobile)
isCollapsed Estado colapsado (desktop)

Events

Evento Descripcion
splitpaneright:open Overlay abierto
splitpaneright:close Overlay cerrado
splitpaneright:toggle Toggle ejecutado (incluye collapsed state)
splitpaneright:show Panel mostrado (desktop)
splitpaneright:hide Panel ocultado (desktop)
