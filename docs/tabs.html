Con Indicador Animado

Indicador que se desliza al cambiar de tab.

<div x-data="uxTabs({ activeTab: 0 })">
  <div class="ux-tab-bar ux-tab-bar--indicator" x-ref="tabBar">
    <button class="ux-tab-button" :class="{ 'ux-tab-button--selected': isActive(0) }" @click="select(0)">Home</button>
    ...
    <div class="ux-tab-bar__indicator" :style="indicatorStyle"></div>
  </div>
</div>

Componente Alpine

Usa el componente uxTabs para control avanzado con indicador animado.

// Configuracion
uxTabs({
  activeTab: 0,        // Indice del tab activo inicial
  tabs: [],            // Array de tabs (opcional)
  animated: false      // Animacion de contenido
})

// Metodos disponibles
select(index)          // Seleccionar tab por indice
selectByValue(value)   // Seleccionar por valor
next()                 // Siguiente tab
prev()                 // Tab anterior
isActive(index)        // Verificar si tab esta activo
updateIndicator()      // Actualizar posicion del indicador

// Propiedades
activeTab              // Indice activo actual
indicatorStyle         // Estilos CSS del indicador
