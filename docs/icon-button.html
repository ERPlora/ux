Icon Button

Botones circulares o cuadrados que contienen solo un icono. Ideales para toolbars y acciones secundarias.

Basico (Ghost)

Estilo transparente por defecto, ideal para navbars y toolbars.

<button class="ux-icon-button" aria-label="Menu">
  <svg ...></svg>
</button>

API

Clase Descripcion

.ux-icon-button--filled Fondo solido
.ux-icon-button--outline Solo borde
.ux-icon-button--soft Fondo tenue
.ux-icon-button--xs/sm/lg/xl Tamanos
.ux-icon-button--square/rounded Formas
.ux-icon-button--badge Punto de notificacion
.ux-icon-button--loading Estado de carga

Accesibilidad

Siempre incluye aria-label para describir la accion del boton.

<button class="ux-icon-button" aria-label="Close dialog">
  <svg ...></svg>
</button>

// Para toggle buttons:
<button class="ux-icon-button"
        aria-label="Favorite"
        aria-pressed="true">
  ...
</button>
