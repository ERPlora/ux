Scroll to Top Button

Boton flotante para volver al inicio. Aparece automaticamente al hacer scroll.

<!-- Con Alpine.js -->
<div x-data="uxScrollTop({ threshold: 300 })"
     class="ux-scroll-top"
     :class="{ 'ux-scroll-top--visible': visible }"
     @click="scrollToTop()">
  <span class="ux-scroll-top__icon">
    <svg fill="none" stroke="currentColor" viewBox="0 0 24 24">
      <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2"
            d="M5 10l7-7m0 0l7 7m-7-7v18"/>
    </svg>
  </span>
</div>

<!-- Detectar scroll en content -->
<main class="ux-content" @scroll="checkScroll($event.target.scrollTop)">
  ...
</main>

Alpine.js Components

Componentes interactivos disponibles.

uxScrollTop(config)

Boton scroll-to-top con deteccion automatica.

x-data="uxScrollTop({
  threshold: 300  // Pixeles de scroll antes de mostrar
})"

// Propiedades
visible    // Boolean - si el boton es visible

// Metodos
checkScroll(scrollTop)  // Verificar posicion de scroll
scrollToTop()           // Scroll suave al inicio

uxCollapsibleHeader(config)

Header que se oculta al hacer scroll hacia abajo.

x-data="uxCollapsibleHeader({
  threshold: 50  // Pixeles minimos antes de colapsar
})"

// Propiedades
collapsed  // Boolean - si el header esta colapsado

// Metodos
handleScroll(scrollTop)  // Manejar evento de scroll
