Alpine.js Options

Option Type Default Description
currentStepnumber0Current active step
totalStepsnumber0Total steps (auto-detected)
clickablebooleanfalseAllow clicking steps
allowSkipbooleanfalseAllow skipping steps

Methods

Method Description
next()Go to next step
prev()Go to previous step
goToStep(n)Go to specific step
isActive(n)Check if step is active
isCompleted(n)Check if step is completed
isDisabled(n)Check if step is disabled
reset()Reset to first step

Events

Event Detail Description
progresssteps:change{ step }Step changed
progresssteps:next{ step }Moved to next
progresssteps:prev{ step }Moved to previous
progresssteps:reset-Steps reset
