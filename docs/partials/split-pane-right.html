Split Pane Right

Panel lateral derecho con toggle. En desktop el panel es visible y puede colapsarse. En mobile aparece como overlay desde la derecha.

Comportamiento

  • Desktop (>= 992px): Panel visible, puede ocultarse con toggle
  • Mobile (< 992px): Panel como overlay desde la derecha con backdrop

Este es el contenido principal. El panel lateral puede mostrarse u ocultarse con el boton de la barra de navegacion.

API

Metodo/Prop Descripcion
toggle() Toggle panel (desktop: collapse, mobile: overlay)
open() Abrir overlay (mobile)
close() Cerrar overlay (mobile)
show() Mostrar panel (desktop)
hide() Ocultar panel (desktop)
isOpen Estado del overlay (mobile)
isCollapsed Estado colapsado (desktop)

Events

Evento Descripcion
splitpaneright:open Overlay abierto
splitpaneright:close Overlay cerrado
splitpaneright:toggle Toggle ejecutado (incluye collapsed state)
splitpaneright:show Panel mostrado (desktop)
splitpaneright:hide Panel ocultado (desktop)

HTMX + Django

Ejemplos de integracion con HTMX y Django para cargar contenido dinamico en el panel lateral.

Lista con detalle en panel

Al hacer click en un item de la lista, se carga el detalle en el panel lateral derecho.

<!-- templates/products/list.html -->
<div class="ux-split-pane-right"
     x-data="uxSplitPaneRight()"
     :class="containerClasses">

  <div class="ux-split-pane-right__backdrop" @click="close()"></div>

  <div class="ux-split-pane-right__main">
    <nav class="ux-navbar ux-navbar--bordered">
      <div class="ux-navbar__start">
        <span class="ux-navbar__title">Productos</span>
      </div>
      <div class="ux-navbar__end">
        <button class="ux-split-pane-right__toggle" @click="toggle()">
          <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
            <rect x="3" y="3" width="18" height="18" rx="2" ry="2"/>
            <line x1="15" y1="3" x2="15" y2="21"/>
          </svg>
        </button>
      </div>
    </nav>

      {% for product in products %}
      <li class="ux-list__item"

          hx-target="#product-detail"

          @htmx:after-request="open()">
        {{ product.name }}

  </div>

  <div class="ux-split-pane-right__side">
    <nav class="ux-navbar ux-navbar--bordered">

      <p style="color: var(--ux-text-secondary);">Selecciona un producto para ver detalles</p>

Vista Django

# views.py
from django.shortcuts import render, get_object_or_404
from .models import Product

def product_list(request):
    products = Product.objects.all()
    return render(request, 'products/list.html', {'products': products})

def product_detail(request, pk):
    product = get_object_or_404(Product, pk=pk)
    # Partial template para HTMX
    return render(request, 'products/_detail.html', {'product': product})

URLs

# urls.py
from django.urls import path
from . import views

app_name = 'products'

urlpatterns = [
    path('', views.product_list, name='list'),
    path('<int:pk>/', views.product_detail, name='detail'),
    path('<int:pk>/edit/', views.product_edit, name='edit'),
    path('<int:pk>/delete/', views.product_delete, name='delete'),
    path('create/', views.product_create, name='create'),
]
