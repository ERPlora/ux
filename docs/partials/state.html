State Components

Componentes para estados de UI: empty, error, success, warning, offline. Estilo iOS minimalista con iconos SVG y animaciones sutiles.

Empty State

Cuando no hay datos para mostrar.

No hay elementos

Aun no tienes ningun elemento guardado. Comienza agregando uno nuevo.

<div class="ux-state ux-empty-state">
  <div class="ux-state__icon">
    <svg>...</svg>
  </div>
  <div class="ux-state__content">
    <h4 class="ux-state__title">No hay elementos</h4>
    <p class="ux-state__description">Descripcion del estado vacio</p>
  </div>
  <div class="ux-state__actions">

  </div>
</div>

Error State

Cuando ocurre un error.

Error al cargar

No pudimos cargar los datos. Por favor verifica tu conexion e intenta de nuevo.

Success State

Cuando una accion se completa exitosamente.

Pago completado

Tu pedido ha sido procesado correctamente. Recibiras un email de confirmacion.

Warning State

Para advertencias o situaciones que requieren atencion.

Stock bajo

Algunos productos de tu carrito tienen stock limitado. Completa tu compra pronto.

Offline State

Cuando el usuario pierde conexion.

Sin conexion

Parece que no tienes conexion a internet. Algunas funciones no estaran disponibles.

Search Empty State

Cuando una busqueda no tiene resultados.

Sin resultados

No encontramos resultados para "zapatos rojos". Intenta con otros terminos.

HTMX + Django

Ejemplos de manejo de estado con HTMX y Django para renderizado del lado del servidor.

Estado vacio con carga HTMX

Cargar contenido dinamico y mostrar estado vacio si no hay resultados.

<!-- Template: products/list.html -->
<div id="product-list"
     hx-get="/products/"
     hx-trigger="load"
     hx-indicator="#loading-state">

  <!-- Estado de carga inicial -->
  <div id="loading-state" class="ux-state ux-loading-state htmx-indicator">
    <div class="ux-state__icon ux-state__icon--animated">
      <div class="ux-spinner"></div>
    </div>
    <div class="ux-state__content">
      <h4 class="ux-state__title">Cargando productos...</h4>
    </div>
  </div>
</div>

Vista Django con estado vacio

# views.py
from django.shortcuts import render

def product_list(request):
    products = Product.objects.filter(active=True)
    return render(request, 'products/partials/list.html', {
        'products': products,
        'is_empty': not products.exists()
    })

Detector de conexion con Alpine.js

<!-- Combinar con Alpine.js para detectar estado offline -->
<div x-data="{ online: navigator.onLine }"
     @online.window="online = true"
     @offline.window="online = false">

  <template x-if="!online">
    <div class="ux-state ux-offline-state ux-state--card">
      <div class="ux-state__icon">
        <svg viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="1.5">
          <line x1="1" y1="1" x2="23" y2="23"/>
          <path d="M16.72 11.06A10.94 10.94 0 0 1 19 12.55"/>
          <path d="M5 12.55a10.94 10.94 0 0 1 5.17-2.39"/>
          <line x1="12" y1="20" x2="12.01" y2="20"/>
        </svg>
      </div>
      <div class="ux-state__content">
        <h4 class="ux-state__title">Sin conexion</h4>
        <p class="ux-state__description">Algunas funciones no estan disponibles.</p>
      </div>
    </div>
  </template>

  <!-- Contenido principal -->
  <div x-show="online">
    <!-- ... -->
  </div>
</div>
