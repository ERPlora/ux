HTMX + Django

Ejemplos de indicadores de estado dinamicos con HTMX y Django.

Estado de servidor con polling

Actualiza el estado del servidor cada 5 segundos.

<!-- Template: server_status.html -->
<div hx-get="{% url 'server_status' server.id %}"
     hx-trigger="load, every 5s"
     hx-swap="innerHTML">
  <span class="ux-status-indicator ux-status-indicator--{{ server.status }} ux-status-indicator--pulse">
    <span class="ux-status-indicator__dot"></span>
    <span class="ux-status-indicator__label">{{ server.status_display }}</span>
  </span>
</div>

Vista Django

# views.py
from django.http import HttpResponse
from django.template.loader import render_to_string

def server_status(request, server_id):
    server = Server.objects.get(id=server_id)
    # status puede ser: success, warning, danger, info
    status_map = {
        'running': 'success',
        'degraded': 'warning',
        'stopped': 'danger',
        'starting': 'info',
    }
    context = {
        'server': server,
        'status': status_map.get(server.status, 'info'),
        'status_display': server.get_status_display(),
    }
    html = render_to_string('partials/server_status_indicator.html', context)
    return HttpResponse(html)

Estado de usuario en tiempo real

Actualiza el estado de presencia del usuario con SSE.

<!-- Avatar con estado de presencia dinamico -->
<div style="position: relative; display: inline-block;"
     hx-ext="sse"
     sse-connect="{% url 'user_presence_stream' user.id %}"
     sse-swap="presence">
  <div class="ux-avatar">{{ user.initials }}</div>
  <span class="ux-status-indicator ux-status-indicator--{{ user.presence }} ux-status-indicator--ring ux-status-indicator--dot-only"
        style="position: absolute; bottom: 2px; right: 2px;">
    <span class="ux-status-indicator__dot"></span>
  </span>
</div>

Lista de servicios con estado

<!-- Template: services_list.html -->
<div hx-get="{% url 'services_status' %}"
     hx-trigger="load, every 10s"
     hx-swap="innerHTML">
  {% for service in services %}
  <div style="display: flex; align-items: center; justify-content: space-between; padding: var(--ux-space-sm);">
    <span>{{ service.name }}</span>
    <span class="ux-status-indicator ux-status-indicator--{{ service.status_class }} ux-status-indicator--filled">
      <span class="ux-status-indicator__dot"></span>
      <span class="ux-status-indicator__label">{{ service.status_label }}</span>
    </span>
  </div>
  {% endfor %}
</div>

Cambiar estado con accion

<!-- Cambiar estado de tarea -->
<div id="task-{{ task.id }}-status">
  <span class="ux-status-indicator ux-status-indicator--{{ task.status }}"
        hx-post="{% url 'toggle_task_status' task.id %}"
        hx-target="#task-{{ task.id }}-status"
        hx-swap="innerHTML"
        style="cursor: pointer;">
    <span class="ux-status-indicator__dot"></span>
    <span class="ux-status-indicator__label">{{ task.status_display }}</span>
  </span>
</div>
