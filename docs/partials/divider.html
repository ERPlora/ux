HTMX + Django

Uso de dividers en templates Django con contenido dinamico cargado via HTMX.

Template Django

<!-- templates/partials/product_list.html -->
{% for product in products %}
  <div class="product-item">
    <h4>{{ product.name }}</h4>
    <p>{{ product.description }}</p>
  </div>
  {% if not forloop.last %}
    <div class="ux-divider ux-divider--inset ux-divider--none"></div>
  {% endif %}
{% empty %}
  <p>No hay productos disponibles.</p>
{% endfor %}

Carga dinamica con HTMX

<!-- Lista con carga dinamica -->
<div id="product-list"
     hx-get="/api/products/"
     hx-trigger="load"
     hx-swap="innerHTML">
  <div class="ux-spinner"></div>
</div>

<!-- Seccion con divider de texto -->
<div class="ux-divider">Productos destacados</div>

<div id="featured-products"
     hx-get="/api/products/featured/"
     hx-trigger="load"
     hx-swap="innerHTML">
</div>

Vista Django

# views.py
from django.shortcuts import render

def product_list(request):
    products = Product.objects.all()

    if request.headers.get('HX-Request'):
        return render(request, 'partials/product_list.html', {
            'products': products
        })

    return render(request, 'products.html', {
        'products': products
    })
