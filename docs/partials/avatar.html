Avatares dinamicos con Django.

Template Django - Avatar de usuario

<!-- templates/partials/avatar_usuario.html -->
{% if usuario.avatar %}
  <div class="ux-avatar ux-avatar--lg">
    <img src="{{ usuario.avatar.url }}" alt="{{ usuario.nombre }}">
    {% if usuario.is_online %}
      <span class="ux-avatar__status ux-avatar__status--online"></span>
    {% else %}
      <span class="ux-avatar__status ux-avatar__status--offline"></span>
    {% endif %}
  </div>
{% else %}
  <div class="ux-avatar ux-avatar--lg ux-avatar--{{ usuario.color_avatar }}">
    {{ usuario.iniciales }}
  </div>
{% endif %}

Vista Django

# views.py
from django.shortcuts import render

def obtener_avatar(request, usuario_id):
    usuario = Usuario.objects.get(id=usuario_id)
    return render(request, 'partials/avatar_usuario.html', {
        'usuario': usuario
    })

def miembros_proyecto(request, proyecto_id):
    proyecto = Proyecto.objects.get(id=proyecto_id)
    miembros = proyecto.miembros.all()[:5]
    total = proyecto.miembros.count()

    return render(request, 'partials/avatar_grupo.html', {
        'miembros': miembros,
        'restantes': total - 5 if total > 5 else 0
    })

Grupo de avatares dinamico

<!-- templates/partials/avatar_grupo.html -->
<div class="ux-avatar-group"
     hx-get="{% url 'miembros_proyecto' proyecto.id %}"
     hx-trigger="load"
     hx-swap="innerHTML">
  {% for miembro in miembros %}
    <div class="ux-avatar">
      {% if miembro.avatar %}
        <img src="{{ miembro.avatar.url }}" alt="{{ miembro.nombre }}">
      {% else %}
        {{ miembro.iniciales }}
      {% endif %}
    </div>
  {% endfor %}
  {% if restantes > 0 %}
    <div class="ux-avatar-group__more">+{{ restantes }}</div>
  {% endif %}
</div>

Integracion con HTMX (Basico)

Avatares dinamicos cargados desde el servidor.

<!-- Avatar que carga imagen del servidor -->
<div class="ux-avatar ux-avatar--lg ux-avatar--loading"
     hx-get="/api/user/123/avatar"
     hx-trigger="load"
     hx-swap="outerHTML">
</div>

<!-- Respuesta del servidor -->
<div class="ux-avatar ux-avatar--lg">
  <img src="/uploads/avatar-123.jpg" alt="Usuario">
  <span class="ux-avatar__status ux-avatar__status--online"></span>
</div>
<!-- Avatar clickable que abre perfil -->
<div class="ux-avatar ux-avatar--lg ux-avatar--clickable"
     hx-get="/api/user/123/profile"
     hx-target="#user-profile-modal"
     hx-trigger="click">
  <img src="/avatar.jpg" alt="">
</div>

<div id="user-profile-modal"></div>
<!-- Grupo de avatares cargado dinamicamente -->
<div class="ux-avatar-group"
     hx-get="/api/project/456/members"
     hx-trigger="load">
  <!-- Loading placeholders -->
  <div class="ux-avatar ux-avatar--loading"></div>
  <div class="ux-avatar ux-avatar--loading"></div>
  <div class="ux-avatar ux-avatar--loading"></div>
</div>

</div>
