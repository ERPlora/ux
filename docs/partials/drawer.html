Drawer

Panel lateral deslizable (sidebar modal). Ideal para navegacion, filtros y paneles de configuracion.

Basico (Left)

Drawer desde la izquierda con backdrop.

<div x-data="uxDrawer()">
  <button @click="open()">Open</button>

  <template x-teleport="body">
    <div class="ux-drawer-backdrop" :class="backdropClasses">
      <div class="ux-drawer">
        ...content...
      </div>
    </div>
  </template>
</div>

Drawer con contenido cargado via HTMX

Carga el contenido del drawer cuando se abre.

<!-- Boton que abre el drawer y carga contenido -->
<div x-data="uxDrawer()" @drawer:open="$refs.content.innerHTML = ''">
  <button class="ux-button ux-color-primary"
          @click="open()"
          hx-get="/api/user/profile/"
          hx-target="#drawer-content"
          hx-trigger="click">
    Ver Perfil
  </button>

  <template x-teleport="body">
    <div class="ux-drawer-backdrop" :class="backdropClasses" @click.self="onBackdropClick()">
      <div class="ux-drawer ux-drawer--lg" :class="drawerClasses">
        <div class="ux-drawer__header">
          <h3 class="ux-drawer__title">Perfil de Usuario</h3>
          <button class="ux-drawer__close" @click="close()">&times;</button>
        </div>
        <div class="ux-drawer__content" id="drawer-content" x-ref="content">
          <div class="ux-skeleton" style="height: 200px;"></div>
        </div>
      </div>
    </div>
  </template>
</div>

Django View

# views.py
from django.shortcuts import render

def user_profile(request):
    user = request.user
    return render(request, 'partials/user_profile.html', {
        'user': user,
        'orders_count': user.orders.count(),
        'favorites_count': user.favorites.count(),
    })

API

Clase Descripcion
.ux-drawer-backdrop Contenedor con backdrop
.ux-drawer Panel lateral
.ux-drawer--right Posicion derecha
.ux-drawer--sm/lg/xl/full Anchos
.ux-drawer--glass Efecto glass morphism
.ux-drawer__header Cabecera con titulo
.ux-drawer__content Area de contenido scrollable
.ux-drawer__footer Pie con acciones

Alpine.js API

Opcion/Metodo Descripcion
position 'left' | 'right' (default: 'left')
closeOnBackdrop Cerrar al click en backdrop (default: true)
closeOnEscape Cerrar con tecla Escape (default: true)
lockScroll Bloquear scroll del body (default: true)
open() Abrir drawer
close() Cerrar drawer
toggle() Alternar estado
isOpen Estado actual

Events

Evento Descripcion
drawer:open Drawer abierto
drawer:close Drawer cerrado
