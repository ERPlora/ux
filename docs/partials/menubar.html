Menubar

Horizontal menu bar with dropdown menus, similar to desktop application menus. Supports keyboard navigation, submenus, and various item types.

Basic Menubar

Standard menubar with dropdown menus. Hover or click to open.

Click on menu items to toggle dropdowns. Once open, hover to switch between menus.

<nav class="ux-menubar">
  <div class="ux-menubar__menu">
    <button class="ux-menubar__trigger" data-state="open">File</button>
    <div class="ux-menubar__content" data-state="open">
      <button class="ux-menubar__item">
        New File
        <span class="ux-menubar__shortcut">Ctrl+N</span>
      </button>
      <div class="ux-menubar__separator"></div>
      <button class="ux-menubar__item">Save</button>
    </div>
  </div>
</nav>

With React/Radix

Example using Radix UI Menubar.

import * as Menubar from '@radix-ui/react-menubar';

<Menubar.Root className="ux-menubar">
  <Menubar.Menu>
    <Menubar.Trigger className="ux-menubar__trigger">File</Menubar.Trigger>
    <Menubar.Portal>
      <Menubar.Content className="ux-menubar__content" sideOffset={5}>
        <Menubar.Item className="ux-menubar__item">
          New File <span className="ux-menubar__shortcut">Ctrl+N</span>
        </Menubar.Item>
        <Menubar.Separator className="ux-menubar__separator" />
        <Menubar.Sub>
          <Menubar.SubTrigger className="ux-menubar__sub-trigger">
            Share
            <span className="ux-menubar__sub-arrow">&#x203A;</span>
          </Menubar.SubTrigger>
          <Menubar.SubContent className="ux-menubar__sub-content">
            <Menubar.Item className="ux-menubar__item">Email</Menubar.Item>
          </Menubar.SubContent>
        </Menubar.Sub>
      </Menubar.Content>
    </Menubar.Portal>
  </Menubar.Menu>
</Menubar.Root>
