API Alpine - Toast Manager

Componente completo para gestionar multiples toasts.

<div x-data="uxToastManager()">
  <!-- Botones para mostrar toasts -->
  <button @click="show('Mensaje')">Default</button>
  <button @click="success('Guardado!')">Success</button>
  <button @click="error('Error!')">Error</button>
  <button @click="warning('Cuidado!')">Warning</button>
  <button @click="info('Info!')">Info</button>

  <!-- Contenedor de toasts -->
  <div class="ux-toast-container">
    <template x-for="toast in toasts" :key="toast.id">
      <div class="ux-toast"
           :class="'ux-toast--' + toast.type"
           x-show="toast.visible"
           x-transition>
        <span x-text="toast.message"></span>
        <button @click="dismiss(toast.id)">X</button>
      </div>
    </template>
  </div>
</div>

Integracion con HTMX

HTMX es la tecnologia principal para mostrar toasts desde respuestas del servidor.

Toast desde respuesta de formulario

<!-- Formulario que muestra toast al guardar -->
<form hx-post="/api/settings"
      hx-target="#toast-container"
      hx-swap="beforeend">
  <input type="text" name="name" class="ux-input">
  <button type="submit" class="ux-button ux-color-primary">Guardar</button>
</form>

<!-- Contenedor fijo para toasts -->
<div id="toast-container" class="ux-toast-container ux-toast-container--bottom"></div>

<!-- Servidor devuelve toast con auto-cierre -->
<div class="ux-toast ux-toast--success ux-toast--visible"
     x-data="{ show: true }"
     x-show="show"
     x-init="setTimeout(() => { show = false; setTimeout(() => $el.remove(), 300) }, 3000)">
  <span class="ux-toast__message">Configuracion guardada</span>
</div>

Toast de exito/error desde respuesta Django

# views.py
from django.http import HttpResponse
from django.template.loader import render_to_string

def contact_create(request):
    form = ContactForm(request.POST or None)

    if form.is_valid():
        form.save()
        # Devolver toast de exito
        return HttpResponse(render_to_string('partials/toast.html', {
            'message': 'Contacto guardado correctamente',
            'type': 'success'
        }))

    # Devolver toast de error
    return HttpResponse(render_to_string('partials/toast.html', {
        'message': 'Error al guardar el contacto',
        'type': 'danger'
    }), status=400)

Vista Django con toast OOB

# views.py - Toast con Out-of-Band swap
from django.http import HttpResponse
from django.template.loader import render_to_string

def product_create(request):
    form = ProductForm(request.POST or None)

    if form.is_valid():
        product = form.save()

        # Contenido principal (fila de tabla)
        row_html = render_to_string('partials/product_row.html', {
            'product': product
        })

        # Toast OOB
        toast_html = render_to_string('partials/toast_oob.html', {
            'message': f'Producto "{product.name}" creado',
            'type': 'success'
        })

        return HttpResponse(row_html + toast_html)

    return HttpResponse(render_to_string('partials/toast_oob.html', {
        'message': 'Error en el formulario',
        'type': 'danger'
    }), status=400)

Mixin reutilizable para vistas

# mixins.py
from django.template.loader import render_to_string

class ToastMixin:
    """Mixin para agregar toasts a respuestas HTMX"""

    def toast_response(self, message, toast_type='success', extra_html=''):
        """Genera respuesta con toast"""
        toast_html = render_to_string('partials/toast_oob.html', {
            'message': message,
            'type': toast_type
        })
        return HttpResponse(extra_html + toast_html)

    def success_toast(self, message, extra_html=''):
        return self.toast_response(message, 'success', extra_html)

    def error_toast(self, message, extra_html=''):
        return self.toast_response(message, 'danger', extra_html)

# Uso en vista
class ProductCreateView(ToastMixin, View):
    def post(self, request):
        form = ProductForm(request.POST)
        if form.is_valid():
            product = form.save()
            row = render_to_string('partials/product_row.html', {'product': product})
            return self.success_toast(f'Producto creado', row)
        return self.error_toast('Error en el formulario')

API Alpine

Metodo Descripcion
show(message, options) Muestra un toast con mensaje y opciones
success(message) Toast de exito
error(message) Toast de error
warning(message) Toast de advertencia
info(message) Toast informativo
dismiss(id) Cierra un toast especifico
dismissAll() Cierra todos los toasts
setPosition(pos) Cambia la posicion del contenedor
containerClass Clase CSS dinamica para el contenedor

Posiciones disponibles

top, bottom, center, top-start, top-end, bottom-start, bottom-end, center-start, center-end
