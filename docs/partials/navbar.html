Uso Basico (3 Slots)

Estructura con 3 slots: start, center, end. El titulo siempre queda centrado aunque los slots esten vacios.

Titulo

    </div>

Posicionamiento (Glass por Defecto)

Los navbars con --fixed y --sticky incluyen efecto glass automaticamente (iOS 26 Liquid Glass). No necesitas agregar la clase --glass.

Glass Automatico 
<!-- Sticky: glass automatico (iOS 26 Liquid Glass) -->
<nav class="ux-navbar ux-navbar--sticky">...</nav>

<!-- Fixed: glass automatico (iOS 26 Liquid Glass) -->
<nav class="ux-navbar ux-navbar--fixed">...</nav>

<!-- Con sombra (usar al hacer scroll) -->
<nav class="ux-navbar ux-navbar--sticky ux-navbar--shadow">...</nav>

<!-- Safe area para iOS notch -->
<nav class="ux-navbar ux-navbar--safe-area">...</nav>

<!-- NOTA: No necesitas --glass, ya esta incluido automaticamente
     en --fixed y --sticky para comportamiento iOS nativo -->

Componente Alpine (Colapsable)

Usar el componente Alpine uxNavbar para controlar el colapso del large title y la sombra al hacer scroll.

     x-data="uxNavbar({ scrollThreshold: 100 })"
     :class="{ 'ux-navbar--collapsed': collapsed, 'ux-navbar--shadow': showShadow }"
     @scroll.window="handleScroll($event.target.documentElement.scrollTop)">

      <button class="ux-navbar__button" @click="goBack()">Back</button>
    </div>
    <div class="ux-navbar__center">

    </div>
    <div class="ux-navbar__end">...</div>
  </div>

HTMX + Django

Ejemplos de integracion con HTMX y Django para navbars dinamicos.

Navbar con Contador de Notificaciones Dinamico

El badge de notificaciones se actualiza automaticamente cada 30 segundos o cuando se dispara un evento.

Mi App 
<!-- Navbar con badge de notificaciones dinamico -->
<nav class="ux-navbar ux-navbar--sticky">
  <div class="ux-navbar__content">
    <div class="ux-navbar__start">
      <button class="ux-navbar__button ux-navbar__button--icon">
        <svg>...</svg>
      </button>
    </div>
    <div class="ux-navbar__center">
      <span class="ux-navbar__title">Mi App</span>
    </div>
    <div class="ux-navbar__end">
      <!-- Badge con polling cada 30s y actualizacion por evento -->
      <button class="ux-navbar__button ux-navbar__button--icon" style="position: relative;"
              hx-get="/api/notifications/count/"
              hx-trigger="load, every 30s, notificationsUpdated from:body"
              hx-target="#notification-badge"
              hx-swap="innerHTML">
        <svg>...</svg>
        <span id="notification-badge"></span>
      </button>
    </div>
  </div>
</nav>

    </div>
  </div>
</div>

Vista Django para Badge de Notificaciones

Endpoint que retorna solo el HTML del badge, optimizado para HTMX.

# views.py
from django.http import HttpResponse
from django.template.loader import render_to_string

def notification_count(request):
    """Retorna el badge de notificaciones para HTMX."""
    count = request.user.notifications.unread().count()

    if count == 0:
        return HttpResponse("")  # Sin badge si no hay notificaciones

    # Badge con numero (max 99+)
    display = "99+" if count > 99 else str(count)
    html = f'''<span class="ux-badge ux-color-danger"
               style="position: absolute; top: 0; right: 0;
                      min-width: 18px; height: 18px; font-size: 11px;">
               {display}
           </span>'''

    return HttpResponse(html)

def user_menu(request):
    """Retorna el contenido del menu de usuario."""
    return render_to_string('partials/user_menu.html', {
        'user': request.user
    }, request=request)
