Hover Card

A card that appears when hovering over a trigger element. Useful for showing user profiles, previews, or additional information.

Delayed Show

Card appears after a delay to prevent accidental triggers.

What is this?

This hover card has a 300ms delay before appearing. This prevents accidental triggers when users move their cursor across the page.

With React/Radix

Example using Radix UI Hover Card.

import * as HoverCard from '@radix-ui/react-hover-card';

<HoverCard.Root>
  <HoverCard.Trigger className="ux-hover-card__trigger">
    @johndoe
  </HoverCard.Trigger>
  <HoverCard.Portal>
    <HoverCard.Content className="ux-hover-card__content" sideOffset={4}>
      <div className="ux-hover-card__header">
        <div className="ux-hover-card__avatar">
          <img src="avatar.jpg" alt="John Doe" />
        </div>
        <div className="ux-hover-card__info">
          <h4 className="ux-hover-card__title">John Doe</h4>
          <p className="ux-hover-card__subtitle">@johndoe</p>
        </div>
      </div>
      <p className="ux-hover-card__description">Bio here...</p>
      <HoverCard.Arrow />
    </HoverCard.Content>
  </HoverCard.Portal>
</HoverCard.Root>

With HTMX

Load card content dynamically on hover.

<div class="ux-hover-card"
     hx-get="/api/user/123/card"
     hx-trigger="mouseenter"
     hx-target="find .ux-hover-card__content"
     hx-swap="innerHTML">
  <a href="/user/123" class="ux-hover-card__trigger">@johndoe</a>
  <div class="ux-hover-card__content">
    <!-- Content loaded via HTMX -->
  </div>
</div>
