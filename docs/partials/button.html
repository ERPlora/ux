Basico

El boton basico usa la clase .ux-button. Sin clase de color, usa primary por defecto.

<button class="ux-button">Default (Primary)</button>
<button class="ux-button ux-color-primary">Primary</button>
<button class="ux-button ux-color-secondary">Secondary</button>
<button class="ux-button ux-color-tertiary">Tertiary</button>

Colores

Usa las clases .ux-color-{color} para aplicar colores.

<button class="ux-button ux-color-success">Success</button>
<button class="ux-button ux-color-warning">Warning</button>
<button class="ux-button ux-color-danger">Danger</button>

Variantes

Outline, clear, y solid (default). Para outline usa .ux-color-{color}--outline.

<button class="ux-button ux-color-primary">Solid</button>
<button class="ux-button ux-button--outline ux-color-primary--outline">Outline</button>
<button class="ux-button ux-button--clear ux-color-primary">Clear</button>

Glass (iOS 26 Liquid Glass)

Boton con efecto de vidrio esmerilado para interfaces modernas. Usa .ux-button--glass.

<button class="ux-button ux-button--glass">Glass Button</button>
<button class="ux-button ux-button--glass ux-button--sm">Small Glass</button>
<button class="ux-button ux-button--glass ux-button--round">Round Glass</button>

Tamanos

Pequeno, mediano (default), y grande.

<button class="ux-button ux-color-primary ux-button--sm">Small</button>
<button class="ux-button ux-color-primary">Medium</button>
<button class="ux-button ux-color-primary ux-button--lg">Large</button>

Formas

Rounded (default), round, y block.

<button class="ux-button ux-color-primary ux-button--round">Round</button>
<button class="ux-button ux-color-primary ux-button--block">Block</button>

Con Iconos

Iconos al inicio, al final, o solo icono.

<button class="ux-button ux-color-primary">
  <svg class="ux-button__icon">...</svg>
  Add Item
</button>

<button class="ux-button ux-color-primary ux-button--icon">
  <svg>...</svg>
</button>

Estados

Disabled y loading.

<button class="ux-button ux-color-primary" disabled>Disabled</button>
<button class="ux-button ux-color-primary ux-button--loading">Loading</button>

Grupo de Botones

Agrupa botones con .ux-button-group.

<div class="ux-button-group">
  <button class="ux-button ux-color-primary">Left</button>
  <button class="ux-button ux-color-primary">Center</button>
  <button class="ux-button ux-color-primary">Right</button>
</div>

HTMX + Django

Botones con HTMX para acciones asincronas en Django.

Template Django

<!-- templates/partials/boton_accion.html -->
<button class="ux-button ux-color-primary"
        hx-post="{% url 'procesar_accion' %}"
        hx-target="#resultado"
        hx-swap="innerHTML"
        hx-indicator="#loading">
  <span class="htmx-indicator" id="loading">
    <span class="ux-spinner ux-spinner--sm"></span>
  </span>
  Procesar
</button>

<div id="resultado"></div>

Vista Django

# views.py
from django.http import HttpResponse
from django.views.decorators.http import require_POST

@require_POST
def procesar_accion(request):
    # Procesar la accion
    resultado = realizar_operacion()

    # Retornar fragmento HTML
    return HttpResponse(f'''
        <div class="ux-badge ux-color-success">
            Completado: {resultado}
        </div>
    ''')
