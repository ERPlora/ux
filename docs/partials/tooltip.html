Tooltips

Tooltips para mostrar información adicional al hacer hover sobre elementos. Útil para explicar iconos, botones y acciones.

Posiciones

Los tooltips pueden aparecer en 4 posiciones: arriba (default), abajo, izquierda y derecha.

Tooltip arriba Tooltip abajo Tooltip izquierda Tooltip derecha 
<!-- Top (default) -->
<span class="ux-tooltip">
  <button class="ux-button">Hover me</button>
  <span class="ux-tooltip__content">Tooltip text</span>
</span>

<!-- Bottom -->
<span class="ux-tooltip ux-tooltip--bottom">...</span>

<!-- Left -->
<span class="ux-tooltip ux-tooltip--left">...</span>

<!-- Right -->
<span class="ux-tooltip ux-tooltip--right">...</span>

Integracion con HTMX

Los tooltips pueden cargar contenido dinamicamente del servidor usando HTMX.

Tooltip con Contenido Dinamico

Cargar contenido del tooltip desde el servidor al hacer hover.

Cargando... Cargando info... 
<!-- Tooltip que carga contenido al hover -->
<span class="ux-tooltip ux-tooltip--rich">
  <button hx-get="/api/user/1/tooltip"
          hx-target="next .ux-tooltip__content"
          hx-trigger="mouseenter"
          hx-swap="innerHTML">
    Ver usuario
  </button>
  <span class="ux-tooltip__content">
    Cargando...
  </span>
</span>

<!-- Cargar solo una vez con "once" -->
<button hx-get="/api/product/info"
        hx-trigger="mouseenter once">
  Info
</button>

HTMX + Django

Ejemplos de integración de tooltips con contenido dinámico usando Django como backend.

Vista Django para Tooltip de Usuario

Endpoint que devuelve un fragmento HTML con información del usuario para el tooltip.

# views.py
from django.shortcuts import render, get_object_or_404
from django.http import HttpResponse
from .models import User

def user_tooltip(request, user_id):
    """Devuelve HTML para tooltip de usuario"""
    user = get_object_or_404(User, pk=user_id)
    return render(request, 'partials/user_tooltip.html', {'user': user})
<!-- templates/partials/user_tooltip.html -->
<div class="ux-tooltip__title">{{ user.get_full_name }}</div>
<div class="ux-tooltip__description">
  <strong>Email:</strong> {{ user.email }}<br>
  <strong>Rol:</strong> {{ user.role }}<br>
  <strong>Miembro desde:</strong> {{ user.date_joined|date:"d M Y" }}
</div>
<!-- En tu template principal -->
<span class="ux-tooltip ux-tooltip--rich ux-tooltip--light">
  <a href="{% url 'user_detail' user.id %}"
     hx-get="{% url 'user_tooltip' user.id %}"
     hx-target="next .ux-tooltip__content"
     hx-trigger="mouseenter once"
     hx-swap="innerHTML">
    {{ user.username }}
  </a>
  <span class="ux-tooltip__content">
    <span class="ux-spinner ux-spinner--sm"></span> Cargando...
  </span>
</span>

Tooltip de Producto con Stock

Mostrar información de stock y disponibilidad al hacer hover sobre un producto.

# views.py
def product_tooltip(request, product_id):
    """Tooltip con info de producto y stock en tiempo real"""
    product = get_object_or_404(Product, pk=product_id)
    stock = product.get_stock_info()

    return render(request, 'partials/product_tooltip.html', {
        'product': product,
        'stock': stock,
        'in_stock': stock.quantity > 0
    })
<!-- templates/partials/product_tooltip.html -->
<div class="ux-tooltip__title">{{ product.name }}</div>
<div class="ux-tooltip__description">
  <strong>SKU:</strong> {{ product.sku }}<br>
  <strong>Precio:</strong> ${{ product.price }}<br>
  {% if in_stock %}
    <span style="color: var(--ux-success);">
      En stock ({{ stock.quantity }} disponibles)
    </span>
  {% else %}
    <span style="color: var(--ux-danger);">Sin stock</span>
  {% endif %}
</div>

Tooltip de Pedido con Detalles

Preview de un pedido con listado de items y totales.

# views.py
def order_preview(request, order_id):
    """Preview de pedido para tooltip"""
    order = get_object_or_404(Order, pk=order_id)

    # Solo los primeros 3 items para el preview
    items = order.items.all()[:3]
    remaining = order.items.count() - 3

    return render(request, 'partials/order_preview.html', {
        'order': order,
        'items': items,
        'remaining': remaining if remaining > 0 else 0
    })
<!-- templates/partials/order_preview.html -->
<div class="ux-tooltip__title">
  Pedido #{{ order.id }}
  <span class="ux-badge ux-badge--sm ux-color-{{ order.status_color }}--soft">
    {{ order.get_status_display }}
  </span>
</div>
<div class="ux-tooltip__description">
  {% for item in items %}
    <div>{{ item.quantity }}x {{ item.product.name|truncatewords:3 }}</div>
  {% endfor %}
  {% if remaining %}
    <div style="color: var(--ux-text-tertiary);">
      +{{ remaining }} items más...
    </div>
  {% endif %}
  <div style="margin-top: 0.5rem; font-weight: 600;">
    Total: ${{ order.total }}
  </div>
</div>

Ayuda Contextual desde Base de Datos

Cargar textos de ayuda dinámicos para formularios.

# models.py
class HelpText(models.Model):
    """Textos de ayuda para formularios"""
    field_name = models.CharField(max_length=100, unique=True)
    title = models.CharField(max_length=200)
    content = models.TextField()

    class Meta:
        verbose_name = "Texto de ayuda"

# views.py
def field_help(request, field_name):
    """Devuelve texto de ayuda para un campo"""
    try:
        help_text = HelpText.objects.get(field_name=field_name)
        return render(request, 'partials/help_tooltip.html', {
            'help': help_text
        })
    except HelpText.DoesNotExist:
        return HttpResponse("Sin ayuda disponible")
<!-- templates/partials/help_tooltip.html -->
{% if help.title %}
  <div class="ux-tooltip__title">{{ help.title }}</div>
{% endif %}
<div class="ux-tooltip__description">{{ help.content }}</div>
<!-- Uso en formulario Django -->
{% for field in form %}
  <div class="ux-input">
    <div style="display: flex; align-items: center; gap: 0.5rem;">
      <label class="ux-input__label" for="{{ field.id_for_label }}">
        {{ field.label }}
      </label>
      <span class="ux-tooltip ux-tooltip--sm">
        <span hx-get="{% url 'field_help' field.name %}"
              hx-target="next .ux-tooltip__content"
              hx-trigger="mouseenter once"
              style="cursor: help; color: var(--ux-text-secondary);">
          <svg width="14" height="14" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
            <circle cx="12" cy="12" r="10"/>
            <path d="M9.09 9a3 3 0 0 1 5.83 1c0 2-3 3-3 3"/>
            <line x1="12" y1="17" x2="12.01" y2="17"/>
          </svg>
        </span>
        <span class="ux-tooltip__content">Cargando...</span>
      </span>
    </div>
    {{ field }}
    {% if field.errors %}
      <div class="ux-input__error">{{ field.errors.0 }}</div>
    {% endif %}
  </div>
{% endfor %}
