Audio Player

Reproductor de audio completo con controles, visualizacion y soporte para playlist.

Player Basico

Reproductor de audio con controles esenciales: play/pause, progreso y tiempo.

<div x-data="uxAudioPlayer({
  src: 'audio.mp3',
  title: 'Song Title',
  artist: 'Artist Name',
  artwork: 'cover.jpg'
})" class="ux-audio-player">
  <!-- Track info, progress, controls... -->
</div>

Con Playlist

Reproductor con soporte para multiples pistas.

Playlist
<div x-data="uxAudioPlayer({
  playlist: [
    { src: 'track1.mp3', title: 'Track 1', artist: 'Artist' },
    { src: 'track2.mp3', title: 'Track 2', artist: 'Artist' },
    { src: 'track3.mp3', title: 'Track 3', artist: 'Artist' }
  ]
})" class="ux-audio-player">
  <!-- Player controls -->

  <!-- Playlist -->
  <div class="ux-audio-player__playlist">
    <template x-for="(track, index) in playlist" :key="index">
      <div class="ux-audio-player__playlist-item"
           :class="{ 'ux-audio-player__playlist-item--active': currentTrackIndex === index }"
           @click="playTrack(index)">
        <span x-text="index + 1"></span>
        <span x-text="track.title"></span>
      </div>
    </template>
  </div>
</div>

HTMX + Django

Integracion con HTMX para cargar pistas dinamicamente.

Player con pista dinamica

<!-- Cargar pista desde el servidor -->
<div x-data="uxAudioPlayer()"
     x-init="$watch('src', () => _loadTrack())"
     hx-get="/api/tracks/current/"
     hx-trigger="load"
     hx-target="this"
     hx-swap="none"
     @htmx:after-request="
       const data = JSON.parse($event.detail.xhr.responseText);
       src = data.src;
       title = data.title;
       artist = data.artist;
       artwork = data.artwork;
     "
     class="ux-audio-player">
  <!-- Player UI -->
</div>

Vista Django para pistas

# views.py
from django.http import JsonResponse

def current_track(request):
    track = Track.objects.filter(user=request.user).last()
    return JsonResponse({
        'src': track.audio_file.url,
        'title': track.title,
        'artist': track.artist.name,
        'artwork': track.cover.url if track.cover else None
    })

def playlist_tracks(request, playlist_id):
    playlist = Playlist.objects.get(id=playlist_id)
    tracks = [{
        'src': t.audio_file.url,
        'title': t.title,
        'artist': t.artist.name,
        'artwork': t.cover.url if t.cover else None
    } for t in playlist.tracks.all()]
    return JsonResponse({'tracks': tracks})

Playlist dinamica con HTMX

<div x-data="uxAudioPlayer({ playlist: [] })"

API Alpine

Opciones de configuracion

Opcion Default Descripcion
src '' URL del archivo de audio
title 'Unknown Track' Titulo de la pista
artist 'Unknown Artist' Nombre del artista
artwork '' URL de la imagen de portada
volume 1 Volumen inicial (0-1)
playbackRate 1 Velocidad de reproduccion
playlist [] Array de pistas para playlist
showWaveform false Mostrar visualizacion waveform
shuffle false Reproduccion aleatoria
repeat 'none' Modo repeticion: 'none', 'one', 'all'

Metodos

Metodo Descripcion
play() Iniciar reproduccion
pause() Pausar reproduccion
toggle() Alternar play/pause
stop() Detener y reiniciar
seek(time) Ir a tiempo especifico (segundos)
seekPercent(percent) Ir a porcentaje especifico
seekForward(seconds) Adelantar N segundos
seekBackward(seconds) Retroceder N segundos
setVolume(value) Establecer volumen (0-1)
toggleMute() Alternar silencio
setPlaybackRate(rate) Establecer velocidad
playTrack(index) Reproducir pista de playlist
nextTrack() Siguiente pista
prevTrack() Pista anterior
toggleShuffle() Alternar modo aleatorio
toggleRepeat() Ciclar modo repeticion

Propiedades reactivas

Propiedad Descripcion
isPlaying Si esta reproduciendo
isLoading Si esta cargando
currentTime Tiempo actual (segundos)
duration Duracion total (segundos)
progress Progreso en porcentaje
currentTimeFormatted Tiempo formateado (m:ss)
durationFormatted Duracion formateada (m:ss)
currentTrack Objeto de pista actual
currentTrackIndex Indice de pista actual
