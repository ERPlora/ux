Sheets (Glass por Defecto)

Bottom sheets, side sheets y action sheets estilo iOS. Componentes overlay que aparecen desde los bordes de la pantalla. Todos incluyen efecto glass automaticamente (iOS 26 Liquid Glass).

Bottom Sheet Basico

Sheet que aparece desde la parte inferior con handle arrastrable.

Click para abrir

Bottom Sheet

Este es un bottom sheet basico con handle arrastrable y boton de cerrar.

Puedes arrastrar hacia abajo para cerrar o hacer click en el backdrop.

<!-- Los sheets tienen glass automatico (iOS 26 Liquid Glass) -->
<div x-data="uxSheet()">
  <button @click="open()">Abrir</button>

  <div class="ux-sheet-backdrop" :class="{ 'ux-sheet-backdrop--open': isOpen }">
    <div class="ux-sheet">
      <div class="ux-sheet__handle"><div class="ux-sheet__handle-bar"></div></div>
      <div class="ux-sheet__header">...</div>
      <div class="ux-sheet__content">...</div>
      <div class="ux-sheet__footer">...</div>
    </div>
  </div>
</div>

     para comportamiento iOS nativo -->

uxSheet(config) - Bottom sheet basico

// Opciones
{
  isOpen: false,           // Estado inicial
  detent: 'medium',        // 'small', 'medium', 'large'
  closeOnBackdrop: true,   // Cierra al click en backdrop
  draggable: true          // Permite arrastrar para cerrar
}

// Metodos
open(detent)    // Abre el sheet (opcional: detent)
close()         // Cierra el sheet
toggle()        // Alterna abierto/cerrado
setDetent(d)    // Cambia el detent

uxActionSheet(config) - Action sheet

// Opciones
{
  title: '',          // Titulo opcional
  message: '',        // Mensaje opcional
  buttons: [],        // Array de botones
  cancelText: 'Cancel'
}

// Metodos
open(options)   // Abre con opciones dinamicas
close()         // Cierra

uxSideMenu(config) - Side menu con gestos

// Opciones
{
  side: 'left',           // 'left' o 'right'
  type: 'overlay',        // 'overlay', 'push', 'reveal'
  swipeEnabled: true,     // Habilita swipe
  swipeThreshold: 50,     // Pixeles para activar
  edgeWidth: 20           // Ancho de zona edge
}

// Metodos
open()          // Abre el menu
close()         // Cierra el menu
toggle()        // Alterna

// Propiedades computadas
menuClasses     // Clases para el menu
backdropClasses // Clases para backdrop
getMenuStyle()  // Estilos durante drag

HTMX + Django

Integracion de sheets con HTMX y Django para cargar contenido dinamico desde el servidor.

Sheet que carga contenido via HTMX

El contenido del sheet se carga desde el servidor cuando se abre.

<!-- Template Django -->
<div x-data="{ isOpen: false }">
  <button class="ux-button ux-color-primary"
          @click="isOpen = true"
          hx-get="{% url 'user_profile_sheet' %}"
          hx-target="#sheet-content"
          hx-trigger="click">
    Ver Perfil
  </button>

  <div class="ux-sheet-backdrop"
       :class="{ 'ux-sheet-backdrop--open': isOpen }"
       @click.self="isOpen = false">
    <div class="ux-sheet ux-sheet--md">
      <div class="ux-sheet__handle"><div class="ux-sheet__handle-bar"></div></div>
      <div class="ux-sheet__header">
        <h4 class="ux-sheet__title">Perfil de Usuario</h4>
        <button class="ux-sheet__close" @click="isOpen = false">
          <span class="ux-sheet__close-icon">&times;</span>
        </button>
      </div>
      <div id="sheet-content" class="ux-sheet__content">
        <!-- Contenido cargado via HTMX -->
        <div class="ux-spinner"></div>
      </div>
    </div>
  </div>
</div>

Vista Django - Contenido del Sheet

# views.py
from django.shortcuts import render, get_object_or_404
from django.http import HttpResponse

def user_profile_sheet(request):
    """Retorna el contenido del sheet de perfil"""
    user = request.user
    return render(request, 'partials/user_profile_sheet.html', {
        'user': user,
        'stats': {
            'pedidos': user.orders.count(),
            'favoritos': user.favorites.count(),
            'resenas': user.reviews.count(),
        }
    })

def producto_actions(request, producto_id):
    """Retorna las acciones disponibles para un producto"""
    producto = get_object_or_404(Producto, id=producto_id)
    user = request.user

    # Determinar acciones disponibles segun contexto
    actions = []
    if producto.stock > 0:
        actions.append({'icon': 'cart', 'label': 'Agregar al carrito', 'url': f'/cart/add/{producto_id}/'})
    if user.is_authenticated:
        is_favorite = producto in user.favorites.all()
        actions.append({
            'icon': 'heart',
            'label': 'Quitar de favoritos' if is_favorite else 'Agregar a favoritos',
            'url': f'/favorites/toggle/{producto_id}/'
        })
    actions.append({'icon': 'share', 'label': 'Compartir', 'url': f'/share/{producto_id}/'})

    return render(request, 'partials/producto_actions.html', {
        'producto': producto,
        'actions': actions,
    })
