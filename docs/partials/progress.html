Interactivo con Alpine

Progreso

  <div class="ux-progress">
    <div class="ux-progress__bar"
         :style="'width: ' + percent + '%'"></div>
  </div>
  <button @click="increment(10)">+10</button>
  <button @click="decrement(10)">-10</button>
  <button @click="reset()">Reset</button>
  <button @click="complete()">100%</button>
</div>

HTMX + Django

Actualizar progreso via servidor Django.

Template Django - Upload con progreso

<!-- templates/upload/form.html -->
<form hx-post="{% url 'upload_archivo' %}"
      hx-encoding="multipart/form-data"
      hx-target="#resultado">
  <input type="file" name="archivo" required>
  <button type="submit" class="ux-button ux-color-primary">Subir</button>
</form>

<!-- Progreso actualizado via polling -->
<div id="upload-progress"
     hx-get="{% url 'upload_status' %}"
     hx-trigger="every 500ms [uploading]"
     hx-swap="innerHTML">
</div>

<div id="resultado"></div>

Vista Django

# views.py
from django.shortcuts import render
from django.http import HttpResponse
from django.core.cache import cache

def upload_status(request):
    progress = cache.get(f'upload_{request.session.session_key}', 0)

    return render(request, 'partials/progress_bar.html', {
        'progress': progress
    })
