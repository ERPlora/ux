Modal con Bootstrap Grid

El modal usa Bootstrap Grid para controlar el ancho. En movil se muestra como sheet desde abajo.

<!-- Modal con Bootstrap Grid -->
<div class="ux-modal-backdrop">
  <div class="container-fluid">
    <div class="row justify-content-center align-items-center"
         style="min-height: 100vh">
      <div class="col-12 col-md-8 col-lg-5">
        <div class="ux-modal">
          <div class="ux-modal__handle"></div>
          <div class="ux-modal__header">...</div>
          <div class="ux-modal__content">...</div>
          <div class="ux-modal__footer">...</div>
        </div>
      </div>
    </div>
  </div>
</div>

Modal Full Height

Usa .ux-modal--full-height para un modal que ocupe toda la altura de la pantalla.

<!-- Modal full height -->
<div class="ux-modal ux-modal--full-height">
  ...
</div>

Modal Lateral (Side Panels)

Modales que aparecen desde los lados. Usa .ux-modal-backdrop--side + .ux-modal-backdrop--left/right en el backdrop y .ux-modal--left/right en el modal.

<!-- Modal lateral izquierda -->
<div class="ux-modal-backdrop ux-modal-backdrop--side ux-modal-backdrop--left">
  <div class="ux-modal ux-modal--left" style="width: 320px;">
    ...
  </div>
</div>

<!-- Modal lateral derecha -->
<div class="ux-modal-backdrop ux-modal-backdrop--side ux-modal-backdrop--right">
  <div class="ux-modal ux-modal--right" style="width: 400px;">
    ...
  </div>
</div>

Sin Cierre al Click en Backdrop

Usa closeOnBackdrop: false para deshabilitar el cierre al hacer click fuera del modal.

<!-- Deshabilitar cierre en backdrop -->
<div x-data="uxModal({ closeOnBackdrop: false })">
  ...
</div>

Modal CSS Only

Este modal funciona solo con CSS usando el checkbox hack.

<!-- Checkbox oculto controla el estado -->
<input type="checkbox" id="modal-1" class="ux-modal-toggle">
<label for="modal-1" class="ux-button">Abrir</label>

<div class="ux-modal-backdrop">
  <!-- Usa label for="modal-1" para cerrar -->
</div>

API Alpine

Config / Metodo Descripcion
closeOnBackdrop Boolean (default: true) - Cerrar al click en backdrop
closeOnEscape Boolean (default: true) - Cerrar con tecla Escape
isOpen Boolean - Estado del modal
open() Abre el modal
close() Cierra el modal
toggle() Alterna el estado

Clases CSS

Clase Descripcion
.ux-modal-backdrop Contenedor overlay fijo
.ux-modal-backdrop--open Estado visible
.ux-modal El modal en si
.ux-modal--full-height Modal altura completa
.ux-modal-backdrop--side Backdrop para modales laterales
.ux-modal-backdrop--left Alinea modal a la izquierda
.ux-modal-backdrop--right Alinea modal a la derecha
.ux-modal--left Modal lateral izquierda (slide-in)
.ux-modal--right Modal lateral derecha (slide-in)
.ux-modal--glass Variante glass (iOS 26)
.ux-modal__handle Handle arrastrable (solo movil)
.ux-modal__header Cabecera con titulo y close
.ux-modal__content Contenido scrollable
.ux-modal__content--centered Contenido centrado
.ux-modal__footer Pie con botones
