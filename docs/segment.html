Segment

Control segmentado estilo iOS con indicador animado. Ideal para alternar entre vistas o filtros.

<script src="components/ux-segment.js"></script>

Uso Basico

Segment con botones simples. El indicador se mueve al boton seleccionado.

<div class="ux-segment">
  <div class="ux-segment__indicator"></div>
  <button class="ux-segment-button ux-segment-button--selected">
    <span class="ux-segment-button__label">Todos</span>
  </button>
  <button class="ux-segment-button">
    <span class="ux-segment-button__label">Activos</span>
  </button>
  <button class="ux-segment-button">
    <span class="ux-segment-button__label">Completados</span>
  </button>
</div>

Block (Ancho Completo)

El modificador --block hace que el segment ocupe todo el ancho disponible.

<div class="ux-segment ux-segment--block">
  ...
</div>

Alpine.js Component

Usa el componente uxSegment() para manejo automatico del indicador animado.

<div x-data="uxSegment({
  value: 'photos',
  options: [
    { value: 'photos', label: 'Fotos' },
    { value: 'videos', label: 'Videos' },
    { value: 'albums', label: 'Albums' }
  ]
})" class="ux-segment" x-ref="segment">
  <div class="ux-segment__indicator" :style="indicatorStyle"></div>
  <template x-for="option in options" :key="option.value">
    <button class="ux-segment-button"
            :class="{ 'ux-segment-button--selected': isSelected(option.value) }"
            @click="select(option.value)">
      <span class="ux-segment-button__label" x-text="option.label"></span>
    </button>
  </template>
</div>

uxSegment(config)

x-data="uxSegment({
  value: 'option1',           // Valor inicial seleccionado
  options: [                  // Array de opciones
    { value: 'opt1', label: 'Label 1' },
    { value: 'opt2', label: 'Label 2', disabled: true }
  ],
  disabled: false             // Deshabilitar todo el segment
})"

// Propiedades
value           // Valor actual seleccionado
options         // Array de opciones
disabled        // Boolean - si esta deshabilitado
indicatorStyle  // Object - estilos para el indicador
selectedOption  // Object - opcion seleccionada actual
selectedIndex   // Number - indice de la opcion seleccionada

// Metodos
select(value)       // Seleccionar una opcion
isSelected(value)   // Boolean - si el valor esta seleccionado
updateIndicator()   // Recalcular posicion del indicador
