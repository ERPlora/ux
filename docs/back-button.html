Componente Alpine

Uso con Alpine.js para navegacion automatica.

<div x-data="uxBackButton({ text: 'Inicio', defaultHref: '/' })">
  <button class="ux-back-button"
          @click="handleClick($event)">
    <span class="ux-back-button__icon" x-html="iconSvg"></span>
    <span class="ux-back-button__text" x-text="text"></span>
  </button>
</div>

HTMX + Django

Integracion con HTMX para navegacion SPA-like con historial del navegador.

Product Detail 
<!-- Navegacion HTMX con historial -->
<button class="ux-back-button"
        hx-get="/products/"
        hx-target="#main-content"
        hx-push-url="true"
        hx-swap="innerHTML transition:true">
  <span class="ux-back-button__icon">...</span>
  <span class="ux-back-button__text">Products</span>
</button>

<!-- Con URL dinamica desde Django -->
<a class="ux-back-button"
   hx-get="{% url 'category_list' %}"
   hx-target="#main-content"
   hx-push-url="true">
  <span class="ux-back-button__icon">...</span>
  <span class="ux-back-button__text">{{ parent_category.name }}</span>
</a>

<!-- Restaurar scroll al navegar atras -->
<button class="ux-back-button"
        hx-get="/orders/"
        hx-target="#main-content"
        hx-push-url="true"
        hx-on::after-settle="window.scrollTo(0, 0)">
  <span class="ux-back-button__icon">...</span>
  <span class="ux-back-button__text">Orders</span>
</button>

Ejemplo de vista Django con partial template:

# views.py
from django.shortcuts import render

def product_detail(request, pk):
    product = get_object_or_404(Product, pk=pk)
    context = {
        'product': product,
        'back_url': request.META.get('HTTP_HX_CURRENT_URL', '/products/'),
        'back_text': 'Products',
    }

    # Retornar partial si es request HTMX
    if request.headers.get('HX-Request'):
        return render(request, 'products/_detail.html', context)

    return render(request, 'products/detail.html', context)
<!-- templates/products/_detail.html -->

API Alpine

Propiedad/Metodo Descripcion
text Texto del boton (default: 'Back')
href URL destino (si se omite, usa history.back())
defaultHref URL fallback si no hay historial
disabled Deshabilita el boton
collapsed Estado colapsado del texto
goBack() Navega hacia atras
handleClick(event) Handler de click
collapse() Colapsa el texto
expand() Expande el texto
toggle() Alterna colapso/expansion
iconSvg SVG del icono de flecha
