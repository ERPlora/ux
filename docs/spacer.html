Flex Spacer

Expande para llenar el espacio disponible en contenedores flex.

Logo

  <span>Logo</span>
  <div class="ux-spacer ux-spacer--flex"></div>
  <button>Login</button>
</div>

HTMX + Django

Uso de spacers en layouts dinamicos con HTMX y Django.

<!-- Django template con espaciado consistente -->
<div class="ux-content">
  <div class="ux-spacer ux-spacer--navbar-safe"></div>

  <!-- Lista de items cargados con HTMX -->
  <div id="items-container"
       hx-get="{% url 'items_list' %}"
       hx-trigger="load">
    {% include 'partials/loading.html' %}
  </div>

  <div class="ux-spacer ux-spacer--tabbar-safe"></div>
</div>

<!-- Partial: items con espaciado entre ellos -->
{% for item in items %}
  <div class="ux-card">{{ item.name }}</div>
  {% if not forloop.last %}
    <div class="ux-spacer ux-spacer--md"></div>
  {% endif %}
{% endfor %}

<!-- Header con flex spacer -->
<header style="display: flex; align-items: center;">
  <h1>{{ page_title }}</h1>
  <div class="ux-spacer ux-spacer--flex"></div>
  <button class="ux-button"
          hx-post="{% url 'create_item' %}"
          hx-target="#items-container"
          hx-swap="beforeend">
    Agregar
  </button>
</header>

API

Clase Descripcion
.ux-spacer--flex Flex grow para llenar espacio
.ux-spacer--xs/sm/md/lg/xl/2xl/3xl Altura fija vertical
.ux-spacer--h-xs/sm/md/lg/xl/2xl/3xl Ancho fijo horizontal
.ux-spacer--navbar/toolbar/tabbar Altura de componentes nav
.ux-spacer--safe-top/bottom/left/right Safe area insets
.ux-spacer--navbar-safe/tabbar-safe Componente + safe area
.ux-spacer--hide-mobile Ocultar en mobile (<768px)
.ux-spacer--sm-mobile Reducir a sm en mobile
