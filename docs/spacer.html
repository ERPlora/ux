  <div class="ux-spacer ux-spacer--flex"></div>

</div>

API

Clase Descripcion
.ux-spacer--flex Flex grow para llenar espacio
.ux-spacer--xs/sm/md/lg/xl/2xl/3xl Altura fija vertical
.ux-spacer--h-xs/sm/md/lg/xl/2xl/3xl Ancho fijo horizontal
.ux-spacer--navbar/toolbar/tabbar Altura de componentes nav
.ux-spacer--safe-top/bottom/left/right Safe area insets
.ux-spacer--navbar-safe/tabbar-safe Componente + safe area
.ux-spacer--hide-mobile Ocultar en mobile (<768px)
.ux-spacer--sm-mobile Reducir a sm en mobile
