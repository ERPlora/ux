Action Sheet Responsive

Usa clases CSS con media queries para controlar el ancho del action sheet en diferentes pantallas.

Action Sheets con diferentes anchos

Action Sheet Responsive

~42% en desktop, ~58% en tablet, 100% en movil

Action Sheet Estrecho

~33% en desktop

<!-- Define clases responsive en tu CSS -->
<style>
  @media (min-width: 768px) {
    .ux-action-sheet--col-md-7 { max-width: 58.333%; margin-left: auto; margin-right: auto; }
    .ux-action-sheet--col-md-6 { max-width: 50%; margin-left: auto; margin-right: auto; }
  }
  @media (min-width: 992px) {
    .ux-action-sheet--col-lg-5 { max-width: 41.666%; }
    .ux-action-sheet--col-lg-4 { max-width: 33.333%; }
  }
</style>

<!-- Usa las clases en el action sheet -->
<div class="ux-action-sheet ux-action-sheet--col-md-7 ux-action-sheet--col-lg-5">
  ...
</div>

Componentes Alpine

uxSheet(config) - Bottom sheet basico

// Opciones
{
  isOpen: false,           // Estado inicial
  detent: 'medium',        // 'small', 'medium', 'large'
  closeOnBackdrop: true,   // Cierra al click en backdrop
  draggable: true          // Permite arrastrar para cerrar
}

// Metodos
open(detent)    // Abre el sheet (opcional: detent)
close()         // Cierra el sheet
toggle()        // Alterna abierto/cerrado
setDetent(d)    // Cambia el detent

uxActionSheet(config) - Action sheet

// Opciones
{
  title: '',          // Titulo opcional
  message: '',        // Mensaje opcional
  buttons: [],        // Array de botones
  cancelText: 'Cancel'
}

// Metodos
open(options)   // Abre con opciones dinamicas
close()         // Cierra

uxSideMenu(config) - Side menu con gestos

// Opciones
{
  side: 'left',           // 'left' o 'right'
  type: 'overlay',        // 'overlay', 'push', 'reveal'
  swipeEnabled: true,     // Habilita swipe
  swipeThreshold: 50,     // Pixeles para activar
  edgeWidth: 20           // Ancho de zona edge
}

// Metodos
open()          // Abre el menu
close()         // Cierra el menu
toggle()        // Alterna

// Propiedades computadas
menuClasses     // Clases para el menu
backdropClasses // Clases para backdrop
getMenuStyle()  // Estilos durante drag
