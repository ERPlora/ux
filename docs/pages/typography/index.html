Headings

Los headings h1-h6 siguen la escala tipográfica de iOS. Se aplican automáticamente dentro de .ux-app.

Headings con Texto

Los headings crean jerarquía visual clara con el contenido.

Escala Tipográfica

Variables CSS disponibles para tamaños de texto. Usan clamp() para escalado fluido.

Referencia

Heading Variable Peso Equivalente iOS
h1 --ux-font-size-3xl 700 (bold) Large Title
h2 --ux-font-size-2xl 700 (bold) Title 1
h3 --ux-font-size-xl 600 (semibold) Title 2
h4 --ux-font-size-lg 600 (semibold) Title 3
h5 --ux-font-size-md 600 (semibold) Headline
h6 --ux-font-size-base 600 (semibold) Subheadline

Variables de Line Height

Variable Valor Uso
--ux-line-height-tight 1.2 Headings
--ux-line-height-normal 1.4 Texto compacto
--ux-line-height-relaxed 1.6 Texto largo / accesible
