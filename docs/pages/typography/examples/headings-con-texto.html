Bienvenido a la aplicación

Los headings siguen la escala tipográfica de iOS, con tamaños fluidos que se adaptan al viewport.

Sección principal

Usa h2 para las secciones principales de tu página. El peso bold ayuda a crear jerarquía visual.

Subsección

Los h3 usan peso semibold, más sutil que h1/h2 pero claramente diferenciado del texto normal.

Detalle

Ideal para títulos dentro de cards, paneles o listas agrupadas.

Etiqueta

Funciona como headline para grupos de contenido compacto.

Nota secundaria

El nivel más pequeño, equivalente al subheadline de iOS.
