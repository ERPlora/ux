Aspect Ratio

Maintains a consistent aspect ratio for images, videos, and other content. Uses the modern CSS aspect-ratio property.

Common Ratios

Pre-defined aspect ratios for common use cases.

1:1 Square

--square

16:9 Video

--video / --16-9

4:3

--4-3

3:2

--3-2

<div class="ux-aspect-ratio ux-aspect-ratio--square">
  <img src="image.jpg" alt="Square image">
</div>

<div class="ux-aspect-ratio ux-aspect-ratio--video">
  <video src="video.mp4"></video>
</div>

More Ratios

Additional aspect ratios for special layouts.

--21-9

3:4 Portrait

--portrait

Golden Ratio

--golden (1.618:1)

With Images

Images automatically fill the aspect ratio container.

Object Fit Modes

Control how content fits within the aspect ratio.

Cover (default)

--contain

--fill (stretched)

<!-- Contain: Shows full image with letterboxing -->
<div class="ux-aspect-ratio ux-aspect-ratio--video ux-aspect-ratio--contain">
  <img src="portrait.jpg" alt="Portrait">
</div>

<!-- Fill: Stretches to fill (may distort) -->
<div class="ux-aspect-ratio ux-aspect-ratio--video ux-aspect-ratio--fill">
  <img src="image.jpg" alt="Stretched">
</div>

With Video

Embedded videos maintain proper aspect ratio.

<div class="ux-aspect-ratio ux-aspect-ratio--video">
  <iframe src="https://youtube.com/embed/..." allowfullscreen></iframe>
</div>

Custom Ratio

Set any aspect ratio using CSS variable.

2:1 Custom
5:4 Custom
1.85:1 Cinema
<div class="ux-aspect-ratio ux-aspect-ratio--custom"
     style="--ux-aspect-ratio: 2/1;">
  <img src="banner.jpg" alt="Banner">
</div>

Responsive Variants

Different aspect ratios on mobile.

16:9 on desktop
1:1 on mobile
<!-- Video on desktop, square on mobile -->
<div class="ux-aspect-ratio ux-aspect-ratio--video ux-aspect-ratio--mobile-square">
  <img src="image.jpg" alt="Responsive">
</div>

Image Gallery Grid

Create consistent image grids with aspect ratios.

With React/Radix

Example using Radix UI Aspect Ratio.

import * as AspectRatio from '@radix-ui/react-aspect-ratio';

<AspectRatio.Root ratio={16 / 9} className="ux-aspect-ratio">
  <img src="image.jpg" alt="Landscape" className="w-full h-full object-cover" />
</AspectRatio.Root>
