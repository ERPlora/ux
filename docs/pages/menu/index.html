Menu

Menus desplegables y contextuales con soporte para iconos, atajos de teclado, estados y submenus.

Uso Basico

Menu desplegable con items simples.

Con Iconos y Atajos

Items con iconos a la izquierda y atajos de teclado a la derecha.

Estados de Items

Items seleccionados, deshabilitados, con encabezados de grupo y descripcion.

Posicionamiento

El menu puede abrirse hacia arriba, abajo, izquierda o derecha.

Tamaños

Menus en diferentes tamanos: sm, default, lg.

Action Sheet (Mobile)

En pantallas moviles, el menu puede mostrarse como un sheet desde abajo.

Menu Contextual

Menu que aparece al hacer clic derecho.

Componente Alpine

Usa el componente uxMenu para control completo con navegacion por teclado.

HTMX + Django

Integracion con HTMX y Django para menus dinamicos con acciones del servidor.

Menu con Items Dinamicos

Carga los items del menu desde Django usando HTMX.

Acciones de Menu con HTMX

Cada item del menu ejecuta una accion en el servidor.

Vista Django

Vista que retorna los items del menu basado en permisos del usuario.
