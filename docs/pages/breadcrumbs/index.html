Breadcrumbs

Breadcrumbs para mostrar la ruta de navegacion actual. Permite al usuario entender donde esta y navegar hacia atras.

Uso Basico

Breadcrumb simple con tres niveles de navegacion.

Estilos de Separador

Diferentes estilos de separador: chevron (default), slash, arrow, dot.

Con Iconos

Breadcrumbs con iconos para mayor claridad visual.

Variantes

Estilos visuales alternativos: contained y outlined.

Tamaños

Tres tamanos disponibles: sm, default y lg.

Estados

Estados current (pagina actual) y disabled.

Breadcrumbs Colapsables (Alpine.js)

Para rutas muy largas, colapsa los items intermedios automaticamente con un dropdown.

Responsive

Con --responsive en mobile muestra solo el primero y el ultimo item.

Combinaciones

Los modificadores se pueden combinar entre si.
