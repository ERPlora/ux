Toolbar

Barras de herramientas estilo iOS para acciones, navegacion y controles. Incluye variantes para header, footer y teclado.

Uso Basico

Toolbar con botones de icono y slots para inicio, centro y fin.

Variantes de Color

Diferentes estilos de fondo: default, primary, dark, light, translucent.

Tamanos

Tres tamanos: sm (36px), default (44px), lg (56px).

Grupo de Botones

Botones agrupados con fondo y estilos de seleccion.

Divider y Spacer

Separadores visuales y espaciadores flexibles.

Boton con Badge

Indicador numerico en botones de toolbar.

Footer Toolbar

Toolbar con borde superior para usar en el pie de pagina.

Keyboard Toolbar

Barra que aparece sobre el teclado con botones de accion.

Action Toolbar

Barra de acciones con iconos y etiquetas (estilo iOS share sheet).

Posicionamiento (Glass por Defecto)

Los toolbars con --fixed-top y --fixed-bottom incluyen efecto glass automaticamente (iOS 26 Liquid Glass). No necesitas agregar la clase --glass.

HTMX + Django

Integracion de toolbar con HTMX para acciones dinamicas y feedback visual.

Toolbar con Acciones HTMX

Botones que ejecutan acciones en el servidor sin recargar la pagina.

Botones de Accion con Feedback

Action toolbar con estados de carga y respuestas visuales.

Vista Django

Vistas que procesan las acciones del toolbar y devuelven respuestas parciales.
