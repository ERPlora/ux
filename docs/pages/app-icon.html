App Icon

iOS-style app icons for launcher grids, dashboards y app selectors.

App Icons Basicos

Iconos de aplicacion con diferentes colores. Usa .ux-app-grid para el layout.

Tamanos

Cuatro tamanos disponibles: --sm, default, --lg y --xl.

Variantes

Variantes outline, dashed (para boton "Add") y glass.

Con Badge de Notificacion

Usa .ux-app-icon__badge para mostrar contadores de notificacion.

API Reference

Clases CSS

.ux-app-icon                // Contenedor principal
.ux-app-icon--sm            // Tamano pequeno (44px)
.ux-app-icon--lg            // Tamano grande (64px)
.ux-app-icon--xl            // Tamano extra grande (80px)

// Colores
.ux-app-icon--primary       // Color primario
.ux-app-icon--secondary     // Color secundario
.ux-app-icon--success       // Verde
.ux-app-icon--warning       // Amarillo
.ux-app-icon--danger        // Rojo
.ux-app-icon--dark          // Oscuro
.ux-app-icon--light         // Claro

// Variantes
.ux-app-icon--outline       // Solo borde
.ux-app-icon--dashed        // Borde punteado (Add button)
.ux-app-icon--glass         // Glass morphism

// Elementos
.ux-app-icon__badge         // Badge de notificacion

App Grid

.ux-app-grid                // Grid responsive de iconos
.ux-app-grid--compact       // Grid mas compacto
.ux-app-grid--cols-4        // 4 columnas fijas
.ux-app-grid--cols-5        // 5 columnas fijas
.ux-app-grid--cols-6        // 6 columnas fijas
