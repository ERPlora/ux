Chip

Componentes compactos para etiquetas, filtros y seleccion. Soportan colores, tamanos, iconos y estados interactivos.

Uso Basico

Chip simple con fondo suave. Usa .ux-color-{color}--soft para variantes de color suave.

Colores

Variantes de color para diferentes contextos semanticos. Usa .ux-color-{color}--soft.

Variante Filled (Solido)

Chips con fondo solido para mayor enfasis. Usa .ux-chip--filled con .ux-color-{color}.

Variante Outline

Chips con borde y fondo transparente. Usa .ux-chip--outline con .ux-color-{color}--outline.

Tamaños

Tres tamanos disponibles: sm (24px), default (32px), lg (40px).

Con Icono

Chips con icono al inicio para contexto visual.

Con Avatar

Chips con imagen de avatar para representar usuarios.

Con Boton de Cerrar

Chips removibles con boton de cierre.

Estado Seleccionado

Chips que pueden seleccionarse como filtros.

Chip Interactivo

Chips clickables con hover y efecto de escala.

Estado Deshabilitado

Grupo de Chips

Chips agrupados con seleccion simple.

Grupo de Chips (Seleccion Multiple)

Permite seleccionar multiples chips.

Grupo con Scroll Horizontal

Grupo que hace scroll horizontal en lugar de wrap.

Componentes Alpine.js

// uxChip - Chip individual seleccionable
uxChip() => {
  selected: false,
  toggle()   // Alterna estado
  select()   // Selecciona
  deselect() // Deselecciona
}

// uxChipGroup - Grupo de chips
uxChipGroup({
  multiple: false  // true para seleccion multiple
}) => {
  selected: null | [],     // Valor(es) seleccionado(s)
  isSelected(value)        // Verifica si esta seleccionado
  toggle(value)            // Alterna seleccion
  select(value)            // Selecciona un valor
  clear()                  // Limpia la seleccion
}

Integracion con HTMX

Chips que interactuan con el servidor.

<!-- Filtros que actualizan contenido -->
<div class="ux-chip-group" x-data="uxChipGroup()">
  <span class="ux-chip ux-chip--interactive"
        :class="{ 'ux-chip--selected': isSelected('todos') }"
        @click="toggle('todos')"
        hx-get="/api/articles?filter=todos"
        hx-target="#article-list"
        hx-trigger="click">
    Todos
  </span>
</div>

HTMX + Django

Ejemplo completo de filtros con chips y vistas Django.

Template Django - Filtros de categoria

<!-- templates/productos/lista.html -->
<div class="ux-chip-group ux-chip-group--scroll">
  <span class="ux-chip {% if categoria_activa == 'todos' %}ux-chip--selected{% endif %}"
        hx-get="{% url 'productos_lista' %}?categoria=todos"
        hx-target="#productos-grid"
        hx-swap="innerHTML">
    Todos
  </span>
  {% for cat in categorias %}
  <span class="ux-chip ux-chip--interactive {% if categoria_activa == cat.slug %}ux-chip--selected{% endif %}"
        hx-get="{% url 'productos_lista' %}?categoria={{ cat.slug }}"
        hx-target="#productos-grid"
        hx-swap="innerHTML">
    {{ cat.nombre }}
  </span>
  {% endfor %}
</div>

Vista Django

# views.py
from django.shortcuts import render

def productos_lista(request):
    categoria = request.GET.get('categoria', 'todos')
    categorias = Categoria.objects.all()

    if categoria == 'todos':
        productos = Producto.objects.all()
    else:
        productos = Producto.objects.filter(categoria__slug=categoria)

    context = {
        'productos': productos,
        'categorias': categorias,
        'categoria_activa': categoria,
    }

    # Si es HTMX, solo el grid
    if request.headers.get('HX-Request'):
        return render(request, 'productos/partials/grid.html', context)

    return render(request, 'productos/lista.html', context)
