Básico

El botón básico usa la clase .ux-button. Sin clase de color, usa primary por defecto.

Colores

Usa las clases .ux-color-{color} para aplicar colores.

Variantes

Outline, clear, y solid (default). Para outline usa .ux-color-{color}--outline.

Glass (iOS 26 Liquid Glass)

Botón con efecto de vidrio esmerilado para interfaces modernas. Usa .ux-button--glass.

Tamaños

Pequeño, mediano (default), y grande.

Formas

Rounded (default), round, y block.

Con Iconos

Iconos al inicio, al final, o solo icono.

Estados

Disabled y loading.

Grupo de Botones

Agrupa botones con .ux-button-group.

HTMX + Django

Botones con HTMX para acciones asíncronas en Django.

Template Django

<!-- templates/partials/boton_accion.html -->
<button class="ux-button ux-color-primary"
        hx-post="{% url 'procesar_accion' %}"
        hx-target="#resultado"
        hx-swap="innerHTML"
        hx-indicator="#loading">
  <span class="htmx-indicator" id="loading">
    <span class="ux-spinner ux-spinner--sm"></span>
  </span>
  Procesar
</button>

<div id="resultado"></div>

Vista Django

# views.py
from django.http import HttpResponse
from django.views.decorators.http import require_POST

@require_POST
def procesar_accion(request):
    # Procesar la acción
    resultado = realizar_operacion()

    # Retornar fragmento HTML
    return HttpResponse(f'''
        <div class="ux-badge ux-color-success">
            Completado: {resultado}
        </div>
    ''')

Botón Eliminar con Confirmación

<!-- Botón eliminar con confirmación -->
<button class="ux-button ux-color-danger"
        hx-delete="{% url 'eliminar_item' item.id %}"
        hx-target="closest .item-row"
        hx-swap="outerHTML swap:1s"
        hx-confirm="¿Eliminar este elemento?">
  Eliminar
</button>

URLs Django
