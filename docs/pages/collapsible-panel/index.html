HTMX + Django

Load panel content dynamically from the server when the panel is opened.

Lazy Load Content on First Open

<!-- Panel with lazy-loaded content -->
<div x-data="uxPanel()" @panel:open.once="$refs.content.dispatchEvent(new Event('load'))">
  <div class="ux-panel" :class="{ 'ux-panel--open': isOpen }">
    <button class="ux-panel__header" @click="toggle()">
      <div class="ux-panel__header-content">
        <h3 class="ux-panel__title">User Details</h3>
      </div>
      <svg class="ux-panel__toggle" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
        <polyline points="6,9 12,15 18,9"></polyline>
      </svg>
    </button>
    <div class="ux-panel__body">
      <div class="ux-panel__content">
        <div class="ux-panel__content-inner"
             x-ref="content"
             hx-get="/api/users/123/details/"
             hx-trigger="load"
             hx-swap="innerHTML">
          <div class="ux-skeleton" style="height: 100px;"></div>
        </div>
      </div>
    </div>
  </div>
</div>

Django View (Python)

# views.py
from django.shortcuts import render
from django.http import HttpResponse

def user_details(request, user_id):
    user = get_object_or_404(User, pk=user_id)
    return render(request, 'partials/user_details.html', {'user': user})

Refresh Content on Panel Open

<!-- Always refresh content when panel opens -->
<div x-data="uxPanel()">
  <div class="ux-panel" :class="{ 'ux-panel--open': isOpen }">
    <button class="ux-panel__header"
            @click="toggle(); if(!isOpen) htmx.trigger($refs.stats, 'refresh')">
      <div class="ux-panel__header-content">
        <h3 class="ux-panel__title">Live Statistics</h3>
      </div>
      <svg class="ux-panel__toggle" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
        <polyline points="6,9 12,15 18,9"></polyline>
      </svg>
    </button>
    <div class="ux-panel__body">
      <div class="ux-panel__content">
        <div class="ux-panel__content-inner"
             x-ref="stats"
             hx-get="/api/stats/live/"
             hx-trigger="refresh"
             hx-swap="innerHTML">
          <p style="color: var(--ux-text-secondary);">Click to load statistics...</p>
        </div>
      </div>
    </div>
  </div>
</div>

Alpine.js Component: uxPanel

Option Type Default Description
openbooleanfalseInitial open state
disabledbooleanfalseDisable toggling

uxPanel Methods

Method Description
toggle()Toggle panel open/close
open()Open the panel
close()Close the panel

Alpine.js Component: uxPanelGroup

Option Type Default Description
accordionbooleanfalseOnly one panel can be open at a time

uxPanelGroup Methods

Method Description
toggle(index)Toggle panel at the given index
isOpen(index)Check if panel at the given index is open
openAll()Open all panels (not available in accordion mode)
closeAll()Close all panels

Events

Event Detail Description
panel:open-Panel was opened
panel:close-Panel was closed
panel:toggle{ open: boolean }Panel state changed
