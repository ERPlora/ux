Con Alpine.js (Dinamico)

Usa el componente Alpine para manejar items dinamicos con lazy loading y animaciones.

<div class="ux-masonry ux-masonry--cols-3"
     x-data="uxMasonry({
       items: [
         { src: 'image1.jpg', title: 'Title 1' },
         { src: 'image2.jpg', title: 'Title 2' }
       ],
       columns: 3,
       animated: true
     })">
  <div class="ux-masonry__grid">
    <template x-for="(item, index) in items">
      <div class="ux-masonry__item ux-masonry__item--hover">
        <img :src="item.src" :alt="item.title" loading="lazy">
        <div class="ux-masonry__item-overlay">
          <span class="ux-masonry__item-title" x-text="item.title"></span>
        </div>
      </div>
    </template>
  </div>
</div>

Comportamiento Responsive

El grid se adapta automaticamente a diferentes tamanos de pantalla:

  • > 1200px: Columnas definidas (6, 5 se reducen a 4)
  • 992px - 1200px: 6, 5, 4 columnas se reducen a 3
  • 767px - 992px: 6, 5, 4, 3 columnas se reducen a 2
  • < 480px: Todo se reduce a 1 columna

Usa ux-masonry--cols-2-always para mantener 2 columnas en movil.

<!-- Mantiene 2 columnas incluso en movil -->
<div class="ux-masonry ux-masonry--cols-2-always">...</div>

HTMX + Django

Ejemplo de integracion con Django para galeria de imagenes paginada.

<!-- Template: gallery.html -->
<div class="ux-masonry ux-masonry--cols-4 ux-masonry--animated">
  <div class="ux-masonry__grid" id="gallery-grid">
    {% for image in images %}
      {% include "partials/masonry_item.html" %}
    {% endfor %}
  </div>
</div>

{% if has_next %}
<div hx-get="{% url 'gallery' %}?page={{ next_page }}"
     hx-trigger="revealed"
     hx-target="#gallery-grid"
     hx-swap="beforeend"
     hx-select="#gallery-grid > *">
</div>
{% endif %}
<!-- partials/masonry_item.html -->
<div class="ux-masonry__item ux-masonry__item--hover ux-masonry__item--clickable"
     hx-get="{% url 'image_detail' image.id %}"
     hx-target="#lightbox-content"
     hx-trigger="click">
  <img src="{{ image.thumbnail.url }}"
       alt="{{ image.title }}"
       loading="lazy">
  <div class="ux-masonry__item-overlay">
    <span class="ux-masonry__item-title">{{ image.title }}</span>
  </div>
</div>
# views.py
from django.shortcuts import render
from django.core.paginator import Paginator

def gallery(request):
    images = Image.objects.filter(published=True).order_by('-created_at')
    paginator = Paginator(images, 20)
    page = request.GET.get('page', 1)
    images_page = paginator.get_page(page)

    context = {
        'images': images_page,
        'has_next': images_page.has_next(),
        'next_page': images_page.next_page_number() if images_page.has_next() else None,
    }

    if request.headers.get('HX-Request'):
        return render(request, 'partials/masonry_items.html', context)

    return render(request, 'gallery.html', context)
