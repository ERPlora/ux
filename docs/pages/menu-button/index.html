Menu Button

Botón hamburguesa para navegación. Incluye animación de transformación a "X" y múltiples variantes visuales.

Uso Básico

El botón de menú puede usar líneas CSS o un icono SVG.

Con Animación

La clase .ux-menu-button--open o el atributo aria-expanded="true" transforma las líneas en una "X".

Tamaños

Tres tamaños disponibles: pequeño, default y grande.

Variantes

Diferentes estilos visuales: default, filled, outlined y rounded.

Con Etiqueta

Incluye texto junto al icono para mayor claridad.

Variante Glass

Efecto glass morphism para interfaces modernas.

Referencia de Clases CSS

Clase Descripción
.ux-menu-button Contenedor principal del botón
.ux-menu-button--open Estado abierto (transforma ☰ a ✕)
.ux-menu-button--sm Tamaño pequeño (36px)
.ux-menu-button--lg Tamaño grande (52px)
.ux-menu-button--filled Con fondo sólido
.ux-menu-button--outlined Con borde
.ux-menu-button--rounded Bordes completamente redondeados
.ux-menu-button--glass Efecto glass morphism
.ux-menu-button--labeled Permite agregar etiqueta de texto
.ux-menu-button__icon Contenedor para icono SVG
.ux-menu-button__lines Contenedor para líneas CSS
.ux-menu-button__line Línea individual del hamburguesa
.ux-menu-button__label Texto del botón (con --labeled)

Accesibilidad

Atributo Descripción
aria-label Describe el propósito del botón (ej: "Abrir menú")
aria-expanded Indica si el menú está abierto ("true") o cerrado ("false")
aria-controls ID del elemento que controla (el menú/drawer)
