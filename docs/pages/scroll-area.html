Auto-Hide Scrollbar

Hover over this area to see the scrollbar appear. It will hide when you move your cursor away.

This is useful for cleaner interfaces where you don't want the scrollbar to always be visible.

The scrollbar will also appear when you start scrolling, providing visual feedback.

Utility Class (Any Element)

Apply custom scrollbar to any scrollable element with utility class.

Use .ux-scrollbar utility class on any element with overflow: auto to apply custom scrollbar styling.

This is useful when you don't need the full scroll area component structure.
