KDS Order

Tarjetas de pedido para Kitchen Display System (KDS). Diseñadas para mostrar órdenes en cocinas de restaurantes con estados visuales claros y acciones rápidas.

Uso Básico

Una tarjeta de pedido con información del número de orden, mesa, tiempo transcurrido e items.

Estados del Pedido

Los pedidos tienen diferentes estados visuales: pendiente (amarillo), preparando (azul), listo (verde).

Alerta y Prioridad

Los pedidos urgentes o retrasados muestran un estado de alerta con animación pulsante y badge de prioridad.

Modo Compacto

Versión compacta para pantallas con muchos pedidos o cafeterías con items simples.

Variante Glass

Efecto glass morphism para interfaces modernas.
