Back Button

Boton de navegacion hacia atras estilo iOS. Se integra con navbar y soporta varios estilos.

Uso Basico

Boton, link o solo icono para navegar hacia atras.

Tamaños

Tres tamanos disponibles: sm, default, lg.

Colores

Variantes de color para diferentes fondos.

En Navbar

El back button se integra automaticamente con el navbar.

Colapsable (iOS-style)

El texto puede colapsar/expandir como en iOS al hacer scroll.

Entrada Animada

Animacion de entrada para transiciones de pagina.

Variante Glass (iOS 26 Liquid Glass)

Boton circular con efecto frosted glass, ideal para navbars transparentes o sobre imagenes.

Deshabilitado

Estado deshabilitado via atributo o clase.

Componente Alpine

Uso con Alpine.js para navegacion automatica y estados.

uxBackButton(config)

HTMX + Django

Integracion con HTMX para navegacion SPA-like con historial del navegador.
