Referencia de Clases CSS

Clase Descripción
.ux-currency Contenedor principal (wrapper)
.ux-input Clase del input (usa estilos estándar de ux-input)
.ux-currency--prefix Símbolo al inicio ($100)
.ux-currency--suffix Símbolo al final (100€)
.ux-currency--sm Tamaño pequeño
.ux-currency--lg Tamaño grande
.ux-currency--error Estado de error
.ux-currency--success Estado de éxito
.ux-currency--glass Variante glass morphism
.ux-currency--display Modo display (solo lectura grande)
.ux-currency--align-* Alineación: left, center, right

Opciones Alpine.js

uxCurrencyInput({
  currency: 'USD',        // Código de moneda (USD, EUR, GBP, JPY, etc.)
  value: 0,               // Valor inicial
  symbol: null,           // Símbolo personalizado (auto si null)
  position: null,         // 'prefix' o 'suffix' (auto si null)
  locale: null,           // Locale para formato (auto si null)
  decimals: 2,            // Número de decimales
  min: null,              // Valor mínimo
  max: null,              // Valor máximo
  step: 1,                // Incremento con flechas
  disabled: false,        // Deshabilitar input
  readonly: false         // Solo lectura
})

// Monedas soportadas
// USD, EUR, GBP, JPY, CNY, MXN, BRL, ARS, CLP, COP, PEN, INR, KRW, CAD, AUD, CHF

// Métodos
getValue()                // Obtener valor numérico
setValue(value)           // Establecer valor
getFormattedValue()       // Obtener valor formateado con símbolo
clear()                   // Limpiar valor
increment()               // Incrementar por step
decrement()               // Decrementar por step

// Eventos
@currency:change          // Se dispara al cambiar el valor
                          // detail: { value, formatted, display }
