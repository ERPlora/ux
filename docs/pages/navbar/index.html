Navbar

Barra de navegacion estilo iOS con soporte para titulo, botones, variantes de color y large title colapsable.

Uso Basico (3 Slots)

Estructura con 3 slots: start, center, end. El titulo siempre queda centrado aunque los slots esten vacios.

Solo Titulo (slots vacios)

Variantes de Color

Diferentes estilos de fondo: default, primary, dark, translucent y transparent.

Tamaños

Tres tamanos disponibles: sm (44px), default (56px), lg (64px).

Con Botones de Accion

Botones de texto e iconos en los slots start y end.

Large Title (iOS Style)

Titulo grande estilo iOS con soporte para colapsar al hacer scroll.

Con Searchbar

Navbar con barra de busqueda integrada.

Con Barra de Progreso

Indicador de progreso en la parte inferior del navbar.

Posicionamiento (Glass por Defecto)

Los navbars con --fixed y --sticky incluyen efecto glass automaticamente (iOS 26 Liquid Glass). No necesitas agregar la clase --glass.

Componente Alpine (Colapsable)

Usar el componente Alpine uxNavbar para controlar el colapso del large title y la sombra al hacer scroll.

HTMX + Django

Ejemplos de integracion con HTMX y Django para navbars dinamicos.

Navbar con Contador de Notificaciones Dinamico

El badge de notificaciones se actualiza automaticamente cada 30 segundos o cuando se dispara un evento.

Menu de Usuario con HTMX

Menu dropdown que carga el contenido dinamicamente al abrirse.

Vista Django para Badge de Notificaciones

Endpoint que retorna solo el HTML del badge, optimizado para HTMX.
