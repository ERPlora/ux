Segment

Control segmentado estilo iOS con indicador animado. Ideal para alternar entre vistas o filtros.

Uso Basico

Segment con botones simples. El indicador se mueve al boton seleccionado.

Block (Ancho Completo)

El modificador --block hace que el segment ocupe todo el ancho disponible.

Con Iconos

Los botones pueden incluir iconos junto al texto o solo iconos.

Variantes de Estilo

Diferentes estilos visuales: default, primary, outline y rounded.

Variantes de Color

Colores adicionales para el indicador: secondary, success, danger.

Tamanos

Tres tamanos disponibles: sm, default y lg.

Estados

Botones individuales o el segment completo pueden estar deshabilitados.

Segment con Scroll

Para muchas opciones, usar --scrollable permite scroll horizontal.

Alpine.js Component

Usa el componente uxSegment() para manejo automatico del indicador animado.

uxSegment(config)

HTMX + Django

Integracion con HTMX para filtrado dinamico de contenido sin recargar la pagina.

Segment que Filtra Contenido

Cada boton del segment hace una peticion HTMX para cargar contenido filtrado.

Vista Lista/Grid con Segment

Alternar entre vista de lista y grid usando HTMX.

Vista Django

Ejemplo de vista Django que retorna contenido filtrado segun el parametro.
