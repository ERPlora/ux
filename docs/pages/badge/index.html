HTMX + Django

Badges dinamicos con actualizacion via HTMX.

Template Django - Contador de notificaciones

<!-- templates/partials/notificaciones_badge.html -->
<span id="notificaciones-badge"
      class="ux-badge ux-color-danger ux-badge--sm"
      hx-get="{% url 'contar_notificaciones' %}"
      hx-trigger="every 30s"
      hx-swap="innerHTML">
  {{ count }}
</span>

Vista Django

# views.py
from django.http import HttpResponse

def contar_notificaciones(request):
    count = Notificacion.objects.filter(
        usuario=request.user,
        leida=False
    ).count()

    # Si es 0, retornar vacio para ocultar badge
    if count == 0:
        return HttpResponse('')

    return HttpResponse(str(count))
