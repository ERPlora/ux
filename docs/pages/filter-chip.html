Filter Chip

Chips interactivos para filtrar contenido. Ideales para mostrar filtros activos, categorías seleccionables y opciones de búsqueda.

Uso Básico

Filter chips permiten a los usuarios seleccionar una o múltiples opciones de filtrado.

Con Iconos y Contadores

Los chips pueden incluir iconos de estado y contadores para mostrar información adicional.

Chips Removibles

Los filtros activos pueden incluir un botón para eliminarlos. Útil para mostrar filtros aplicados.

Colores y Variantes

Los chips soportan colores semánticos, variantes soft y outline.

Tamaños

Tres tamaños disponibles para diferentes contextos de uso.

Referencia de Clases CSS

Clase Descripción
.ux-filter-chips Contenedor para agrupar filter chips
.ux-filter-chips__label Etiqueta descriptiva del grupo de filtros
.ux-filter-chips__clear Botón para limpiar todos los filtros
.ux-filter-chip Chip individual de filtro
.ux-filter-chip--active Estado activo/seleccionado
.ux-filter-chip--sm Tamaño pequeño
.ux-filter-chip--lg Tamaño grande
.ux-filter-chip--success Color success (verde)
.ux-filter-chip--warning Color warning (amarillo)
.ux-filter-chip--danger Color danger (rojo)
.ux-filter-chip--soft Variante con fondo suave
.ux-filter-chip--outline Variante solo con borde
.ux-filter-chip__icon Icono dentro del chip
.ux-filter-chip__icon--leading Icono al inicio del chip
.ux-filter-chip__count Contador/badge dentro del chip
.ux-filter-chip__remove Botón para remover el chip
