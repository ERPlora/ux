Panel Layouts

Los panels son layouts de contenido que van dentro de los shells. Cada seccion ocupa un porcentaje fijo del espacio disponible con scroll interno. Cada seccion tiene un ID unico para integracion con HTMX.

Table Wrapper Nuevo

Tabla con header fijo y body scrollable. El scroll horizontal se sincroniza entre header y body. Ideal para tablas con muchas columnas.

Panel Detail con Toggle Mobile Drawer

Main (70%) + aside (30%). En mobile, el aside se oculta y aparece como drawer desde abajo con un boton FAB.

Panel Dashboard

KPIs arriba + graficos (row en desktop, column en mobile) + tabla con scroll. Ideal para dashboards de metricas.

Panel Table

Acciones (56px) + tabla con scroll interno + paginacion (48px). Para listados CRUD. Usa ux-panel-table-wrapper dentro para header fijo.

Panel Split Horizontal Responsive

Division izquierda (35%) / derecha (65%). En mobile se apilan verticalmente con scroll individual.

Panel Settings Responsive

En desktop: menu lateral (220px) + contenido. En mobile: tabs horizontales scrollables + contenido.

Panel Form

Header (64px) + contenido scrollable centrado (max-width: 800px) + acciones (64px).

Panel Grid 2x2 Responsive

4 secciones iguales. En mobile se apilan verticalmente, en desktop grid 2x2.

Panel Cards Responsive

Grid de cards: 1 columna en mobile, 2 en tablet, 3-4 en desktop. Con scroll interno.
