Card

Contenedor visual flexible con multiples variantes y layouts. Soporta header, media, content y footer.

Estructura de Card

Una card se compone de varias partes opcionales. Usa ux-card__content para contenido con padding, o ux-card__body como wrapper general.

<!-- Opcion 1: Sin wrapper -->
<div class="ux-card">
  <div class="ux-card__header">...</div>       <!-- padding propio -->
  <div class="ux-card__media">...</div>        <!-- sin padding -->
  <div class="ux-card__content">...</div>      <!-- padding propio -->
  <div class="ux-card__footer">...</div>       <!-- padding propio -->
</div>

<!-- Opcion 2: Con wrapper para padding uniforme -->
<div class="ux-card">
  <div class="ux-card__body">               <!-- wrapper con padding -->
    <div class="ux-card__header">...</div>
    <div class="ux-card__content">...</div>
    <div class="ux-card__footer">...</div>
  </div>
</div>
