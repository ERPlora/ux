Opciones

Configuracion disponible para el escaner.

Opcion Tipo Default Descripcion
formats array [varios] Formatos de codigo a detectar
continuous boolean false Continuar escaneando despues de detectar
beepOnScan boolean true Reproducir sonido al detectar
vibrate boolean true Vibrar al detectar (movil)
showManualInput boolean true Mostrar campo de entrada manual
fullscreen boolean false Iniciar en pantalla completa

Formatos Soportados

El escaner puede detectar los siguientes formatos de codigo.

EAN-13 EAN-8 UPC-A UPC-E Code 128 Code 39 Code 93 Codabar QR Code Data Matrix

Eventos

Eventos emitidos por el escaner.

  • scanner:detected - Codigo detectado ({ rawValue, format, timestamp })
  • scanner:started - Escaner iniciado
  • scanner:stopped - Escaner detenido
  • scanner:error - Error ({ error })
  • scanner:close - Escaner cerrado

HTMX + Django

Integra el escaner con HTMX para buscar productos por codigo de barras en tiempo real.

Busqueda de producto al escanear

Al detectar un codigo, se hace una peticion HTMX para buscar el producto y mostrarlo.

<!-- Escaner con busqueda HTMX -->
<div x-data="uxBarcodeScanner({ continuous: false })"
     @scanner:detected="
       $refs.barcodeInput.value = $event.detail.rawValue;
       htmx.trigger($refs.barcodeInput, 'barcode-scan');
     ">

  <video class="ux-barcode-scanner__video" x-ref="video"></video>
  <div class="ux-barcode-scanner__overlay">...</div>

  <!-- Input oculto para HTMX -->
  <input type="hidden"
         x-ref="barcodeInput"
         name="barcode"
         hx-get="/api/productos/buscar/"
         hx-trigger="barcode-scan"
         hx-target="#producto-resultado"
         hx-indicator="#producto-loading">
</div>

<!-- Resultado del producto -->
<div id="producto-resultado"></div>
<div id="producto-loading" class="htmx-indicator">
  <div class="ux-spinner"></div>
</div>

Vista Django

Endpoint que busca el producto y devuelve HTML parcial.

# views.py
from django.shortcuts import render
from django.http import HttpResponse
from .models import Producto

def buscar_producto_por_barcode(request):
    barcode = request.GET.get('barcode', '')

    if not barcode:
        return HttpResponse('<p class="ux-text-secondary">Escanea un codigo</p>')

    try:
        producto = Producto.objects.get(codigo_barras=barcode)
        return render(request, 'partials/producto_card.html', {
            'producto': producto
        })
    except Producto.DoesNotExist:
        return HttpResponse(f'''
            <div class="ux-alert ux-alert--warning">
                Producto no encontrado: {barcode}
            </div>
        ''')

Agregar multiples productos (modo continuo)

Para escenarios de inventario donde se escanean muchos productos seguidos.

<!-- Escaner en modo continuo para inventario -->
<div x-data="{ productos: [] }">
  <div class="ux-barcode-scanner"
       x-data="uxBarcodeScanner({ continuous: true, beepOnScan: true })"
       @scanner:detected="
         fetch('/api/productos/buscar/?barcode=' + $event.detail.rawValue)
           .then(r => r.json())
           .then(data => {
             if (data.producto) {
               productos.push(data.producto);
             }
           })
       ">
    <video class="ux-barcode-scanner__video" x-ref="video"></video>
    <div class="ux-barcode-scanner__overlay">...</div>
  </div>

  <!-- Lista de productos escaneados -->
  <ul class="ux-list">
    <template x-for="producto in productos" :key="producto.id">
      <li class="ux-list__item">
        <span x-text="producto.nombre"></span>
        <span class="ux-badge" x-text="producto.codigo_barras"></span>
      </li>
    </template>
  </ul>

  <!-- Guardar inventario -->
  <button class="ux-button ux-color-success"
          @click="
            fetch('/api/inventario/guardar/', {
              method: 'POST',
              headers: { 'Content-Type': 'application/json' },
              body: JSON.stringify({ productos: productos.map(p => p.id) })
            })
          ">
    Guardar inventario
  </button>
</div>

Compatibilidad

El escaner utiliza la API BarcodeDetector que esta disponible en navegadores modernos.

Navegadores soportados:

  • Chrome 83+ (Android, Desktop)
  • Edge 83+
  • Opera 69+
  • Safari 17+ (iOS, macOS) - parcial

Para navegadores no soportados, el campo de entrada manual proporciona una alternativa.
