Referencia de Clases CSS

Clase Descripcion
.ux-skeleton Clase base con animacion shimmer
.ux-skeleton--text Linea de texto (16px height)
.ux-skeleton--title Titulo (24px height)
.ux-skeleton--circle Forma circular (para avatars)
.ux-skeleton--rect Forma rectangular
.ux-skeleton--rounded Bordes muy redondeados
.ux-skeleton--xs Altura extra pequena (8px)
.ux-skeleton--sm Altura pequena (12px)
.ux-skeleton--md Altura mediana (16px)
.ux-skeleton--lg Altura grande (24px)
.ux-skeleton--xl Altura extra grande (32px)
.ux-skeleton--avatar-* Tamanos de avatar (xs/sm/md/lg/xl)
.ux-skeleton--pulse Animacion pulse en vez de shimmer
.ux-skeleton--no-animation Desactiva la animacion
