Loyalty Card

Tarjetas de fidelización para programas de lealtad, membresías y recompensas. Incluye diferentes niveles (tiers) con efectos visuales atractivos.

Uso Básico

Una tarjeta de lealtad con información de puntos, nombre del miembro y fecha de expiración.

Niveles (Tiers)

Diferentes niveles con gradientes distintivos: Bronze, Silver, Gold, Platinum, Diamond y VIP.

Con Barra de Progreso

Muestra el progreso hacia el siguiente nivel con una barra visual.

Modo Compacto

Versión horizontal compacta para listas o espacios reducidos.

Variante Glass

Efecto glass morphism para interfaces modernas.
