Bar Chart

Graficos de barras CSS-only para visualizaciones simples sin dependencias de JavaScript.

Bar Chart Basico

Grafico de barras vertical con etiquetas y valores.

Barras Horizontales

Usa .ux-bar-chart--horizontal para barras horizontales.

Colores

Variantes de color: primary, success, warning, danger.

Variante Compacta (Mini Sparkline)

Usa .ux-bar-chart--compact para un mini grafico estilo sparkline.

API Reference

Clases CSS

.ux-bar-chart                  // Contenedor principal
.ux-bar-chart__item            // Item individual
.ux-bar-chart__bar             // Barra
.ux-bar-chart__value           // Valor encima de la barra
.ux-bar-chart__value-main      // Valor principal
.ux-bar-chart__value-sub       // Valor secundario
.ux-bar-chart__label           // Etiqueta debajo de la barra

// Variantes de orientacion
.ux-bar-chart--horizontal      // Barras horizontales

// Variantes de tamano
.ux-bar-chart--sm              // Altura 160px
.ux-bar-chart--lg              // Altura 320px
.ux-bar-chart--compact         // Mini sparkline (60px)

// Variantes de color
.ux-bar-chart--primary         // Azul
.ux-bar-chart--success         // Verde
.ux-bar-chart--warning         // Amarillo
.ux-bar-chart--danger          // Rojo

// Variantes especiales
.ux-bar-chart--grid            // Con lineas de grid
.ux-bar-chart--glass           // Glass morphism

Variables CSS

--ux-bar-chart-height: 240px;
--ux-bar-chart-gap: 1rem;
--ux-bar-chart-bar-radius: 6px;
--ux-bar-chart-bar-color: var(--ux-primary);
--ux-bar-chart-bar-gradient: linear-gradient(to top, var(--ux-primary), var(--ux-primary-tint));

Ejemplo de Uso

<div class="ux-bar-chart">
  <div class="ux-bar-chart__item">
    <div class="ux-bar-chart__value">
      <div class="ux-bar-chart__value-main">$2.4k</div>
    </div>
    <div class="ux-bar-chart__bar" style="height: 60%;"></div>
    <div class="ux-bar-chart__label">Ene</div>
  </div>
  <!-- mas items... -->
</div>
