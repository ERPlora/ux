Toggle Group

Group of toggle buttons where one or more can be selected. Similar to iOS Segment Control or Radix Toggle Group.

Basic Toggle Group

Single selection toggle group with data-state attribute.

<div class="ux-toggle-group" role="group">
  <button class="ux-toggle-group__item" data-state="off">Left</button>
  <button class="ux-toggle-group__item" data-state="on">Center</button>
  <button class="ux-toggle-group__item" data-state="off">Right</button>
</div>

Outline Variant

Toggle group with visible borders.

Primary & Soft Colors

Color variants for active state.

Segment Control (iOS Style)

iOS-style segmented control with sliding indicator.

<div class="ux-toggle-group ux-toggle-group--segment">
  <button class="ux-toggle-group__item" data-state="on">All</button>
  <button class="ux-toggle-group__item">Active</button>
  <button class="ux-toggle-group__item">Completed</button>
</div>

Multiple Selection

Allow selecting multiple items (like a toolbar).

Icon Only

Compact toggle group with icons only.

Sizes

Small, default, and large sizes.

Vertical Layout

Vertical toggle group for sidebar filters.

Full Width

Toggle group that spans full container width.

Glass Variant

Glassmorphism effect for modern UI.

Spaced Items

Gap between individual items.

With HTMX

Server-side state management with HTMX.

<div class="ux-toggle-group ux-toggle-group--outline" role="group">
  <button class="ux-toggle-group__item" data-state="on"
          hx-post="/api/view/list" hx-target="#content">List</button>
  <button class="ux-toggle-group__item"
          hx-post="/api/view/grid" hx-target="#content">Grid</button>
</div>
