Onscreen Keyboard

Teclado virtual multi-idioma con layouts personalizables para aplicaciones web, POS y tablets. Soporta idiomas europeos con scripts latinos y cirilicos.

API Reference

Opciones

uxOnscreenKeyboard({
  languages: ['en', 'es', 'fr'],  // Idiomas habilitados
  defaultLanguage: 'en',           // Idioma inicial
  showLanguageSelector: true,      // Mostrar selector de idioma
  showPreview: true,               // Mostrar preview de tecla
  targetSelector: '#my-input',     // Selector del input objetivo
  customLayouts: {},               // Layouts personalizados
  onInput: (key) => {},            // Callback al insertar caracter
  onEnter: (value) => {},          // Callback al presionar Enter
  onClose: () => {}                // Callback al cerrar
})

Metodos

open()                // Abrir el teclado
close()               // Cerrar el teclado
toggle()              // Alternar abrir/cerrar
loadLayout(langCode)  // Cargar layout por codigo
selectLanguage(code)  // Cambiar idioma
pressKey(key)         // Simular pulsacion de tecla
insertCharacter(char) // Insertar caracter en input
toggleShift()         // Alternar Shift/Caps Lock

Eventos

@osk:open      // Teclado abierto
@osk:close     // Teclado cerrado
@osk:input     // Caracter insertado - detail: { key }
@osk:enter     // Tecla Enter presionada - detail: { value }
@osk:backspace // Tecla Backspace presionada
@osk:language  // Idioma cambiado - detail: { language, layout }

Variables CSS

Personaliza el aspecto del teclado con estas variables CSS.

.ux-onscreen-keyboard {
  --ux-osk-bg: var(--ux-surface-secondary);      // Fondo del teclado
  --ux-osk-key-bg: var(--ux-surface);            // Fondo de teclas
  --ux-osk-key-color: var(--ux-text);            // Color de texto
  --ux-osk-key-height: 48px;                     // Altura de teclas
  --ux-osk-key-radius: var(--ux-radius-md);      // Radio de bordes
  --ux-osk-gap: 6px;                             // Espacio entre teclas
  --ux-osk-padding: 8px;                         // Padding del contenedor
  --ux-osk-special-bg: var(--ux-gray-200);       // Fondo teclas especiales
  --ux-osk-active-bg: var(--ux-primary);         // Fondo al presionar
  --ux-osk-active-color: var(--ux-primary-contrast); // Color al presionar
}
