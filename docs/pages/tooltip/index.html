Tooltips

Tooltips para mostrar informacion adicional al hacer hover sobre elementos. Util para explicar iconos, botones y acciones.

Posiciones

Los tooltips pueden aparecer en 4 posiciones: arriba (default), abajo, izquierda y derecha.

Variantes

Diferentes estilos visuales para los tooltips.

Tamaños

Tooltips en diferentes tamanos.

Contenido Multilinea

Tooltips con texto mas largo o contenido enriquecido.

Delay

Retrasar la aparicion del tooltip.

En Iconos

Uso comun: explicar iconos de acciones.

Control Programatico

Control del tooltip con Alpine.js.

Integracion con HTMX

Los tooltips pueden cargar contenido dinamicamente del servidor usando HTMX.

Tooltip con Contenido Dinamico

Cargar contenido del tooltip desde el servidor al hacer hover.

<!-- Tooltip que carga contenido al hover -->
<span class="ux-tooltip ux-tooltip--rich">
  <button hx-get="/api/user/1/tooltip"
          hx-target="next .ux-tooltip__content"
          hx-trigger="mouseenter"
          hx-swap="innerHTML">
    Ver usuario
  </button>
  <span class="ux-tooltip__content">
    Cargando...
  </span>
</span>

<!-- Cargar solo una vez con "once" -->
<button hx-get="/api/product/info"
        hx-trigger="mouseenter once">
  Info
</button>

Preview de Datos

Mostrar preview de datos al hacer hover sobre enlaces o items de lista.

<!-- Preview de pedido en tooltip -->
<span class="ux-tooltip ux-tooltip--rich">
  <button hx-get="/api/orders/1234/preview"
          hx-target="next .ux-tooltip__content"
          hx-trigger="mouseenter">
    <svg>...</svg>
  </button>
  <span class="ux-tooltip__content">
    <!-- El servidor devuelve el preview -->
  </span>
</span>

Tooltip de Ayuda Contextual

Cargar texto de ayuda segun el contexto del formulario.

<!-- Tooltip de ayuda que carga del servidor -->
<span class="ux-tooltip">
  <span hx-get="/api/help/field-name"
        hx-target="next .ux-tooltip__content"
        hx-trigger="mouseenter once"
        style="cursor: help">
    <svg>?</svg>
  </span>
  <span class="ux-tooltip__content">
    Cargando ayuda...
  </span>
</span>

Tooltip con Estado en Tiempo Real

Mostrar estado de un proceso que se actualiza periodicamente.

<!-- Tooltip que se actualiza mientras esta visible -->
<span class="ux-tooltip">
  <button hx-get="/api/server/status"
          hx-target="next .ux-tooltip__content"
          hx-trigger="mouseenter, every 10s[this.matches(':hover')]"
          hx-swap="innerHTML">
    Status
  </button>
  <span class="ux-tooltip__content">
    <!-- Estado actualizado cada 10s mientras hover -->
  </span>
</span>

Tooltip con Accion

Tooltip que incluye acciones que hacen requests HTMX.

<!-- Tooltip con acciones HTMX -->
<span class="ux-tooltip ux-tooltip--rich">
  <button>Item</button>
  <span class="ux-tooltip__content">
    <button hx-post="/api/items/1/edit"
            hx-target="#result">Editar</button>
    <button hx-delete="/api/items/1"
            hx-confirm="Eliminar?">Eliminar</button>
  </span>
</span>
