API - Directivas Alpine

Directiva Modificadores Descripcion
x-swipe .left .right .up .down Detecta swipe en direccion
x-drag .handle:selector Drag & drop con handle opcional
x-pull-refresh - Pull to refresh para listas
x-long-press .{n}ms Long press con duracion configurable
x-pinch - Pinch zoom (solo touch)
x-tap .double Single o double tap

API - uxSwipeItem()

Componente Alpine para swipeable list items.

Propiedad/Metodo Descripcion
offset Desplazamiento actual en px
isSwiping true durante swipe activo
openSide 'start', 'end' o null
contentStyle Getter para :style del contenido
openStart() Abre acciones del inicio
openEnd() Abre acciones del final
close() Cierra las acciones 
// Configuracion
uxSwipeItem({
  startActionsWidth: 160,  // Ancho acciones izquierda
  endActionsWidth: 80,     // Ancho acciones derecha
  threshold: 40            // Distancia para activar snap
})

Vista Django (views.py)

# views.py
from django.http import HttpResponse
from django.views.decorators.http import require_http_methods
from .models import Item

@require_http_methods(["DELETE"])
def delete_item(request, item_id):
    try:
        item = Item.objects.get(id=item_id)
        item.delete()
        # Retornar vacio para que HTMX elimine el elemento
        return HttpResponse("")
    except Item.DoesNotExist:
        return HttpResponse("Item no encontrado", status=404)

URLs Django (urls.py)

# urls.py
from django.urls import path
from . import views

urlpatterns = [
    path('api/items//', views.delete_item, name='delete_item'),
]

Atributos HTMX utiles

Atributo Descripcion
hx-delete Envia DELETE request al endpoint
hx-target Elemento a reemplazar con la respuesta
hx-swap="outerHTML" Reemplaza el elemento completo (ideal para eliminar)
hx-swap="outerHTML swap:200ms" Espera 200ms antes de swap (para animaciones)
hx-confirm Muestra dialogo de confirmacion antes de enviar
