Tabs

Sistema de tabs estilo iOS/Ionic con soporte para iconos, badges, indicador animado y bottom tabs.

Uso Basico

Tab bar con botones y contenido.

Con Iconos

Tabs con iconos arriba del texto (layout por defecto).

Layouts de Icono

Diferentes posiciones para el icono: top, start, end, icon-only, label-only.

Bottom Tab Bar (iOS Style - Glass por Defecto)

Barra de tabs fija en la parte inferior con iconos y labels. Incluye efecto glass automaticamente (iOS 26 Liquid Glass). No necesitas agregar la clase --glass.

Variantes de Color

Tab bars con diferentes colores de fondo.

Estilo Pills

Tabs con estilo de pastillas redondeadas.

Tabs Scrollables

Cuando hay muchos tabs, se hace scroll horizontal.

Con Badges

Tabs con badges de notificacion.

Tamaños

Tab bars en diferentes tamanos: sm, default, lg.

Con Indicador Animado

Indicador que se desliza al cambiar de tab.

Integracion con HTMX

HTMX es la tecnologia principal para cargar contenido de tabs desde el servidor.

Cargar contenido al click

Cada tab carga su contenido desde el servidor al ser seleccionado.

Cargar al iniciar + click

El primer tab carga al iniciar y los demas al click.

Tabs con hx-boost para SPA-like

Navegacion tipo SPA sin JavaScript adicional.

Bottom tabs con navegacion

Barra inferior de navegacion que carga vistas.

Actualizar badge desde servidor

Contadores de notificacion que se actualizan automaticamente.

Tabs con parametros

Enviar datos adicionales al servidor.

HTMX + Django

Ejemplos de integracion con Django para tabs dinamicos con HTMX.

Tabs que cargan contenido via HTMX

Template Django con tabs que cargan partials desde el servidor.

Navegacion SPA-like con HTMX
